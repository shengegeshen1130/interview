## Q2: 考虑五局三胜和三局两胜的情况，哪种更公平

**答案：三局两胜更公平（对弱者更有利），五局三胜更有利于强者。**

设甲单局赢的概率为 p（p > 0.5），乙赢的概率为 q = 1-p。

**关键思路**：甲获胜时，最后一局必定是甲赢。所以只需要对"前面的局"做组合计数。

**三局两胜**中甲获胜概率：

甲需要先赢 2 局，最后一局必须甲赢：
- 第2局结束：前1局赢1局 → C(1,1)p·p = p²
- 第3局结束：前2局赢1局输1局 → C(2,1)pq·p = 2p²q

P₃ = p² + 2p²q = p²(1 + 2q) = p²(1 + 2(1-p)) = **p²(3 - 2p)**

**五局三胜**中甲获胜概率：

甲需要先赢 3 局，最后一局必须甲赢：
- 第3局结束：前2局全赢 → C(2,2)p²·p = p³
- 第4局结束：前3局赢2输1 → C(3,2)p²q·p = 3p³q
- 第5局结束：前4局赢2输2 → C(4,2)p²q²·p = 6p³q²

P₅ = p³ + 3p³q + 6p³q² = p³(1 + 3q + 6q²)

代入 q = 1-p 展开：
= p³(1 + 3(1-p) + 6(1-p)²)
= p³(1 + 3 - 3p + 6 - 12p + 6p²)
= **p³(10 - 15p + 6p²)**

**验证**：当 p = 0.5 时：
- P₃ = 0.25 × (3 - 1) = 0.5 ✓
- P₅ = 0.125 × (10 - 7.5 + 1.5) = 0.125 × 4 = 0.5 ✓

当 p = 0.6 时：
- P₃ = 0.36 × 1.8 = 0.648
- P₅ = 0.216 × (10 - 9 + 2.16) = 0.216 × 3.16 = 0.6826

P₅ > P₃，五局三胜更有利于强者。

**结论**：如果双方实力相当（p ≈ 0.5），两种赛制公平性相同。如果实力有差距，局数越多越有利于强者，"三局两胜"对弱者更友好。直觉理解：局数越多，偶然性越小，实力差距越容易体现。

## Q16: 自写rand函数，实现从1到M里等概率随机抽取N个数

**答案：蓄水池抽样（Reservoir Sampling）或 Fisher-Yates 洗牌**

**方法一：Fisher-Yates 部分洗牌（M 已知且不大）**

```python
import random
def sample(M, N):
    arr = list(range(1, M+1))
    for i in range(N):
        j = random.randint(i, M-1)
        arr[i], arr[j] = arr[j], arr[i]
    return arr[:N]
```

**方法二：蓄水池抽样（适用于流式数据）**

```python
import random
def reservoir_sample(stream, N):
    result = []
    for i, item in enumerate(stream):
        if i < N:
            result.append(item)
        else:
            j = random.randint(0, i)
            if j < N:
                result[j] = item
    return result
```

## Q22: 飞机100个座位，100人按顺序上，第一个人醉了随便坐，其他人正常（自己位置被占就随便坐），求第100个人坐到自己位置的概率

**答案：1/2**

这是经典问题。无论多少人，最后一个人坐到自己座位的概率恒为 **1/2**。

**证明思路一（归纳 + 递推）**：

设 f(n) 为 n 个人时最后一人坐对的概率。

第一个人（醉汉）随机坐一个位置，有 n 种等概率选择：
- 坐在1号（自己的座位）→ 后面所有人都坐对 → 最后一人坐对 ✓
- 坐在n号（最后一人的座位）→ 最后一人必然坐不对 ✗
- 坐在第 k 号（2 ≤ k ≤ n-1）→ 第2到k-1号乘客正常坐自己位置。轮到第k人时发现座位被占，第k人等价于成为新的"醉汉"，问题变成 f(n-k+1)

$$f(n) = \frac{1}{n}\left[1 + 0 + \sum_{k=2}^{n-1} f(n-k+1)\right] = \frac{1}{n}\left[1 + \sum_{j=2}^{n-1} f(j)\right]$$

可以验证 f(2) = 1/2，且由归纳法 f(n) = 1/2 对所有 n ≥ 2 成立。

**证明思路二（对称性，更优雅）**：

观察：在整个过程中，最后一个空着的座位，**只可能是1号或100号**。

为什么？如果座位 k（2 ≤ k ≤ 99）是空的，那么第 k 个乘客上机时它一定是空的（因为其他人不会故意坐别人的座位），第 k 人会坐上去。所以中间的座位不可能最后才空出。

因此最后一个座位要么是1号，要么是100号。由对称性（任何随机选择对1号和100号的处理是对称的），两者等概率。

P(100号最后空着) = P(1号最后空着) = **1/2**

## Q23: 50个红球50个白球，放到两个容器里，怎么放令拿出红球的概率最大

**答案：一个容器放1个红球，另一个容器放49红+50白。**

**规则**：先等概率选一个容器（1/2），再从该容器中等概率抽一个球。

**最优策略的直觉**：让一个容器几乎"稳赢"（100%红球），把概率"浪费"降到最低。

**策略**：容器 A 放1个红球，容器 B 放49红+50白。

$$P(\text{红}) = \frac{1}{2} \times \frac{1}{1} + \frac{1}{2} \times \frac{49}{99} = \frac{1}{2} + \frac{49}{198} = \frac{99 + 49}{198} = \frac{148}{198} = \frac{74}{99} \approx \textbf{74.75\%}$$

**为什么不是放25红+25白到每个容器？** 那样 P = 1/2 × 25/50 + 1/2 × 25/50 = 50%，远不如上面的策略。

**为什么1个红球是最优而不是2个或更多？** 容器A放 k 个红球时：
P = 1/2 × 1 + 1/2 × (50-k)/(100-k)。对 k 求导可知 k 越小越好（但至少放1个），所以 k=1 最优。

## Q25: 你有n个筹码，对手有m个筹码，每次赌博赢的人拿走对面一个筹码，没有筹码就输了，问你赢的概率

**答案（公平赌博，双方单次胜率 1/2）：n/(n+m)**

这是经典的**赌徒破产问题（Gambler's Ruin）**。

**推导（p = 1/2 的情况）**：

设你当前有 k 个筹码时最终获胜的概率为 P(k)。边界条件：P(0) = 0, P(n+m) = 1。

每一轮：赢（概率1/2，筹码+1）或输（概率1/2，筹码-1），得到差分方程：

P(k) = (1/2)P(k+1) + (1/2)P(k-1)

即 P(k+1) - P(k) = P(k) - P(k-1)（等差数列），所以 P(k) 是 k 的线性函数。

由边界条件：P(k) = k/(n+m)，你有 n 个筹码时：

**P(n) = n/(n+m)**

**推导（p ≠ 1/2 的情况）**：

差分方程变为：P(k) = p·P(k+1) + q·P(k-1)，设 r = q/p，通解为 P(k) = A + B·rᵏ

代入边界条件 P(0)=0, P(n+m)=1：

$$P(n) = \frac{1-r^n}{1-r^{n+m}} = \frac{1-(q/p)^n}{1-(q/p)^{n+m}}$$

**例子**：你有10个筹码，对手20个，p=1/2 → P = 10/30 = 33.3%。直觉：筹码少的一方处于劣势。

## Q26: 给一个实时更新的数据流，最开始大小为N，从里面需要抽取m个数据，问新加入的数据怎么能够保证仍然是等概率抽样

**答案：蓄水池抽样（Reservoir Sampling）**

当第 i 个数据到来时（i > m）：
1. 以 m/i 的概率选择该数据
2. 如果被选中，随机替换已选 m 个数据中的一个

可以证明每个元素最终被选中的概率都是 m/i（i 为当前总数据量）。

## Q29: 给定n个苹果，其中一个轻，求最优称的次数的期望

**答案：⌈log₃(n)⌉ 次（天平，最坏情况）**

使用天平（三分法）：
- 将 n 个苹果分成三组（尽量等分）
- 两组放天平两边，若平衡则在第三组，否则在轻的那组
- 每次将问题规模缩小到 1/3

最少次数 = ⌈log₃(n)⌉

例如 n = 27 → 3次，n = 9 → 2次。

## Q32: 给定随机函数R，以p概率产生1，1-p产生0。生成随机函数R'，以1/2概率产生0,1

**答案：调用 R 两次**

- 如果 R() 两次结果为 (1, 0)，输出 1
- 如果 R() 两次结果为 (0, 1)，输出 0
- 如果是 (0,0) 或 (1,1)，丢弃，重来

P(1,0) = p(1-p) = P(0,1)，所以输出 0 和 1 的概率相等，都是 1/2。

每轮产生有效结果的概率 = 2p(1-p)，期望轮数 = 1/(2p(1-p))，每轮调用2次 R()。

期望总调用次数 = **2/(2p(1-p)) = 1/(p(1-p))**

当 p=1/2 时，期望调用 4 次。当 p 接近 0 或 1 时效率很低。

## Q35: 一万个黑球一万个白球，每次摸两个，同色放黑球，异色放白球，最后剩一个黑球的概率

**答案：1（必然是黑球）**

**关键：找不变量（invariant）**

分析每种操作对白球数量的影响：

| 操作 | 取出 | 放回 | 白球变化量 |
|------|------|------|-----------|
| 取两黑 | 2黑 | 1黑 | 0（不变）|
| 取两白 | 2白 | 1黑 | -2 |
| 取一黑一白 | 1黑+1白 | 1白 | 0（不变）|

**白球数量每次要么不变，要么减2**，所以白球数量的**奇偶性是不变量**。

初始白球 = 10000（偶数）→ 过程中白球数始终为偶数 → 最后剩1个球时，白球数只能是 0（不能是1，因为1是奇数）。

所以最后一个球**必然是黑球**。P = **1**

**推广**：如果初始白球是奇数（如9999个），最后一个球必然是白球。这个问题的本质是白球数量的奇偶性守恒。

> **面试技巧**：遇到"最后剩什么"类型的问题，第一反应应该是寻找**不变量**（如奇偶性、模运算等）。

## Q59: X以概率p产生1，1-p产生0，利用X等概率生成1到n的数

**答案：**

**步骤一**：用 Q32 的方法将不公平的 X 转化为公平的 0/1 随机变量 Y（概率各 1/2）。

**步骤二**：用 Y 生成 k = ⌈log₂(n)⌉ 位二进制数，得到 [0, 2^k - 1] 的均匀分布。

## Q60: 扔骰子最多两次，第一次后可以选择是否扔第二次，以最后一次结果为准，求最大化期望的策略和期望值

**答案：期望 = 17/4 = 4.25**

**策略分析**：第一次扔出结果 x 后，选择保留 x 或者重扔。重扔的期望 = (1+2+3+4+5+6)/6 = 3.5。

**最优策略**：x ≥ 4 时保留（4, 5, 6 都大于 3.5），x ≤ 3 时重扔（1, 2, 3 都小于 3.5）。

**期望计算**：

$$E = \underbrace{\frac{1}{6}(4) + \frac{1}{6}(5) + \frac{1}{6}(6)}_{\text{保留}} + \underbrace{\frac{3}{6} \times 3.5}_{\text{重扔}}$$

$$= \frac{4+5+6}{6} + \frac{3 \times 3.5}{6} = \frac{15}{6} + \frac{10.5}{6} = \frac{25.5}{6} = \frac{17}{4} = \textbf{4.25}$$

**直觉理解**：相比无选择权时的期望3.5，有一次"后悔重来"的机会能把期望提高到4.25，提升约21%。

## Q67: 七局四胜制比赛打满七场的概率是多少？（双方每局胜率各 50%）

**答案：5/16**

打满7场 = 前6场双方各赢3场，第7场决胜负。

前6场中一方恰好赢3场的概率：C(6,3) × (1/2)^6 = 20/64 = 5/16

$$P(\text{打满7场}) = \binom{6}{3}\left(\frac{1}{2}\right)^6 = \frac{20}{64} = \frac{5}{16} = \textbf{31.25\%}$$

**推广**：2n-1 局 n 胜制打满 2n-1 场的概率 = C(2n-2, n-1) × (1/2)^(2n-2)

## Q69: 如何在半径为1的圆内均匀随机选取一个点？

**答案：不能直接用 (r, θ) 均匀采样**

**错误方法**：r ~ U(0,1), θ ~ U(0,2π)。这样会导致点集中在圆心附近（因为小半径处面积小但采样密度和大半径一样）。

**正确方法一（逆变换法）**：

面积与 r² 成正比，所以 r² 应该均匀分布：

$$r = \sqrt{U}, \quad \theta = 2\pi V$$

其中 U, V ~ U(0,1) 独立。

**推导**：CDF 为 F(r) = P(R ≤ r) = r²（面积比），所以 R = √U。

**正确方法二（拒绝采样）**：

在 [-1,1]² 的正方形内均匀生成 (x, y)，若 x²+y² ≤ 1 则接受，否则拒绝重来。

接受率 = π/4 ≈ 78.5%，效率较高。

## Q71: 已知 rand7() 返回 1-7 等概率随机数，如何构造 rand10() 返回 1-10？

**答案：拒绝采样法**

**核心思想**：用 rand7() 构造一个范围 ≥ 10 的均匀分布，再拒绝多余的。

```
result = (rand7() - 1) × 7 + rand7()   // 产生 1-49 均匀分布
if result <= 40:
    return (result - 1) % 10 + 1       // 映射到 1-10
else:
    重新调用                              // 拒绝 41-49
```

**原理**：
- (rand7()-1) × 7 产生 {0, 7, 14, 21, 28, 35, 42}
- 加上 rand7() 产生 1 到 49 的均匀分布
- 取前40个（可被10整除），每10个映射到同一输出
- 拒绝 41-49，概率 = 9/49，期望调用次数 = 2 × 49/40 ≈ 2.45 次

**推广**：rand_M() → rand_N()，只要能构造出 ≥ N 的均匀分布即可。

## Q81: 买饮料，三个瓶盖可以换一瓶新的。要喝到100瓶饮料，最少需要购买多少瓶？

**答案：67瓶**

**递推思路**：每买3瓶 → 得3个盖 → 换1瓶 → 实际喝4瓶花3瓶钱。

但不完全是这样，因为换来的瓶子也有盖。

**精确计算**：

设购买 n 瓶。
- 直接喝 n 瓶，得 n 个盖
- 第一轮换：⌊n/3⌋ 瓶，剩余 n%3 个盖
- 继续用新盖换...

用方程：设买 n 瓶能喝 f(n) 瓶。

观察规律：每2个瓶盖的"边际贡献"是 1 瓶（因为3→1→再回收，相当于花2个净盖换1瓶）。

所以 f(n) = n + ⌊(n-1)/2⌋（近似），即 n 瓶能喝约 3n/2 瓶。

100 = 3n/2 → n ≈ 67。

**验证**：买67瓶：
- 喝67瓶，67盖
- 67/3 = 22瓶余1盖，喝22瓶，22+1=23盖
- 23/3 = 7瓶余2盖，喝7瓶，7+2=9盖
- 9/3 = 3瓶余0盖，喝3瓶，3盖
- 3/3 = 1瓶，喝1瓶，1盖（不够换了）
- 总计：67+22+7+3+1 = **100瓶** ✓

## Q82: 100人排队，每人戴黑或白帽子，每人能看到前面所有人的帽子但看不到自己的。从后往前每人猜自己帽子颜色（其他人能听到），猜错死。最优策略下最多死几个人？

**答案：最多死1人（最后一个人可能死）**

**最优策略**：

约定：最后一个人（第100人）数前面99人中**黑帽的奇偶性**：
- 如果黑帽数量为**偶数**，他说"白"
- 如果黑帽数量为**奇数**，他说"黑"

（他有50%概率猜错自己的，所以最多死1人）

**为什么前99人能100%存活？**

第99人听到第100人的回答（得知了黑帽奇偶性），看到前98人的帽子，计算自己是什么颜色才能使总奇偶性匹配。以此类推，每个人利用：
1. 初始奇偶信息（第100人提供）
2. 之后每个人的回答（更新奇偶计数）
3. 自己看到的前方帽子

可以100%确定自己的颜色。

**结论**：最优策略保证至少99人存活，第100人有50%概率存活。

## Q96: 已知 rand5() 以等概率返回 1~5 的整数，如何构造等概率返回 1~7 的 rand7()

**答案：拒绝采样（Rejection Sampling）**

**关键思路**：生成一个均匀覆盖 7 的倍数的更大范围，超出范围则丢弃重试。

**标准实现**：

```python
def rand7():
    while True:
        # (rand5()-1)*5 + rand5() 均匀生成 [1, 25]
        n = (rand5() - 1) * 5 + rand5()
        if n <= 21:          # 只保留 1~21（21 = 3×7）
            return (n - 1) % 7 + 1
```

**正确性分析**：

- `(rand5()-1)*5 + rand5()` 生成 [1, 25] 中的每个整数，概率均为 1/25。
- 保留 [1, 21]：共 21 个等概率整数，恰好是 7 的 3 倍。
- P(输出 k) = P(n ∈ {k, k+7, k+14}) = (3 × 1/25) / (21/25) = 1/7 ✓

**期望 rand5() 调用次数**：

每轮接受概率 p = 21/25，每轮调用 rand5() 2 次。

$$E[\text{调用次数}] = 2 \times \frac{25}{21} = \frac{50}{21} \approx 2.38 \text{ 次}$$

> **推广**：用 rand_A() 构造 rand_B()：调用两次生成 [1, A²] 均匀分布，找 A² 内最大的 B 的倍数，超出则拒绝重试。

## Q98: 最优秘书问题（Secretary Problem / Optimal Stopping）

**题目**：n 个候选人按随机顺序面试，面试完立即决定录/不录（不可回头）。应采用什么策略使录到最优候选人的概率最大？

**答案：拒绝前 ⌊n/e⌋ 个人，之后录取第一个比前面所有人都好的候选人。最优成功概率 ≈ 1/e ≈ 36.8%。**

**推导**：

设策略为"跳过前 r 个人作为参照，之后遇到比参照中最优者更好的立即录取"。

成功要求：最优候选人在位置 k（k > r），且前 k-1 人中最优者在前 r 人中。

$$P(\text{成功}) = \sum_{k=r+1}^{n} \frac{1}{n} \cdot \frac{r}{k-1} = \frac{r}{n} \sum_{k=r}^{n-1} \frac{1}{k} \approx \frac{r}{n} \ln\frac{n}{r}$$

令 t = r/n，对 f(t) = t ln(1/t) 求导：

$$f'(t) = \ln\frac{1}{t} - 1 = 0 \implies t = \frac{1}{e}$$

**最优 r = ⌊n/e⌋，最优成功概率 = 1/e ≈ 36.8%。**

例（n=100）：跳过前37人，之后录取第一个"目前最优"的候选人。

> **应用延伸**：在线学习的 Explore-Exploit 权衡、A/B 测试的早停规则，都与最优停止有关。

### P1: 掷骰子最优停止策略 ⭐

**题目**：你可以掷一个公平六面骰子最多 n 次。每次掷出后，必须立即选择：停下来保留当前点数，或继续掷。编写算法计算最优策略下的期望得分。

**关键思路**：逆向归纳 DP。设 `ev` 为还剩若干次可掷时的最优期望值：
- 0 次可掷：`ev = 0`（不能继续）
- 多 1 次时：掷出 i，若 `i > ev` 则停，否则继续 → `ev = mean(max(i, ev) for i in 1..6)`

```python
def optimal_dice(n: int) -> float:
    """Expected score under optimal strategy with at most n rolls"""
    ev = 0.0
    for _ in range(n):
        ev = sum(max(i, ev) for i in range(1, 7)) / 6
    return ev

for k in range(1, 8):
    print(f"n={k}: {optimal_dice(k):.4f}")
# n=1: 3.5000  n=2: 4.2500  n=3: 4.6667
# n=4: 4.9167  n=5: 5.0972  n=6: 5.2257  n=7: 5.3215
```

**规律**：当 `ev < 3.5` 时阈值为第一个 `> ev` 的整数；局数越多阈值越高，极限趋近 6。

### P2: 赌徒破产问题（Gambler's Ruin）

**题目**：赌徒有 k 元，每轮以概率 p 赢 1 元、1-p 输 1 元，目标攒到 n 元（输光破产）。求：(a) 达目标概率；(b) 期望局数。

**关键思路**：递推方程 `P(k) = p·P(k+1) + q·P(k-1)`，边界 `P(0)=0, P(n)=1`。
- 公平游戏 p=0.5：`P(k) = k/n`，`E[T] = k(n-k)`
- 不公平：`P(k) = (1-(q/p)^k) / (1-(q/p)^n)`

```python
def gamblers_ruin_prob(k: int, n: int, p: float) -> float:
    """P(reach n before ruin | start at k)"""
    if abs(p - 0.5) < 1e-9:
        return k / n
    q = 1 - p
    r = q / p
    return (1 - r**k) / (1 - r**n)

def gamblers_ruin_expected(k: int, n: int, p: float) -> float:
    """Expected rounds to finish | start at k"""
    if abs(p - 0.5) < 1e-9:
        return k * (n - k)
    q = 1 - p
    prob = gamblers_ruin_prob(k, n, p)
    return (k - n * prob) / (q - p)

print(gamblers_ruin_prob(50, 100, 0.50))   # 0.500
print(gamblers_ruin_prob(50, 100, 0.48))   # ≪ 0.5，庄家优势很大
print(gamblers_ruin_expected(50, 100, 0.5))  # 2500 rounds
```

### P3: 硬币序列期望等待时间（Markov Chain + 线性方程组）

**题目**：连续抛公平硬币，期望多少次首次看到序列 "HT"？推广：对任意序列（"HH"、"HTH"…）如何编程求解？

**关键思路**：KMP 自动机建状态图，对各状态 `E[i] = 1 + 0.5*E[next_H(i)] + 0.5*E[next_T(i)]`，组成线性方程组 `numpy` 求解。

```python
import numpy as np

def expected_flips_for_pattern(pattern: str) -> float:
    """Expected fair coin flips to see pattern for first time"""
    n = len(pattern)

    def kmp_next(pat, state, char):
        k = state
        while k > 0 and (k >= len(pat) or pat[k] != char):
            k = kmp_fail(pat)[k - 1]
        if k < len(pat) and pat[k] == char:
            k += 1
        return k

    def kmp_fail(pat):
        f = [0] * len(pat)
        k = 0
        for i in range(1, len(pat)):
            while k > 0 and pat[k] != pat[i]:
                k = f[k - 1]
            if pat[k] == pat[i]:
                k += 1
            f[i] = k
        return f

    fail = kmp_fail(pattern)
    # E[i] - 0.5*E[next_H(i)] - 0.5*E[next_T(i)] = 1  for states i in 0..n-1
    A = np.zeros((n, n))
    b = np.ones(n)
    for i in range(n):
        nh = kmp_next(pattern, i, 'H')
        nt = kmp_next(pattern, i, 'T')
        A[i][i] += 1
        if nh < n:
            A[i][nh] -= 0.5
        if nt < n:
            A[i][nt] -= 0.5
    return np.linalg.solve(A, b)[0]

print(f"E[HT]  = {expected_flips_for_pattern('HT'):.1f}")   # 4.0
print(f"E[HH]  = {expected_flips_for_pattern('HH'):.1f}")   # 6.0
print(f"E[HTH] = {expected_flips_for_pattern('HTH'):.1f}")  # 10.0
```

### P5: 最优取牌策略（区间 DP）

**题目**：n 张牌排成一列，面值各异。甲乙轮流从两端取一张，双方均采用最优策略（令自己总分最大）。甲先取，求甲的最终得分。

**关键思路**：区间 DP。`dp(i,j)` = 从区间 [i,j] 先手能获得的最大分。先手取左/右后，剩余分数全部减去对手的最优得分。

```python
from functools import lru_cache

def optimal_card_game(cards: list) -> int:
    n = len(cards)
    prefix = [0] * (n + 1)
    for i, v in enumerate(cards):
        prefix[i + 1] = prefix[i] + v

    @lru_cache(maxsize=None)
    def dp(i: int, j: int) -> int:
        if i > j:
            return 0
        total = prefix[j + 1] - prefix[i]
        return max(total - dp(i + 1, j),   # take left
                   total - dp(i, j - 1))   # take right

    return dp(0, n - 1)

print(optimal_card_game([8, 15, 3, 7]))  # 22
print(optimal_card_game([3, 9, 1, 2]))   # 11
```

### P6: 蒙提霍尔仿真（Monty Hall）

**题目**：三扇门，奖品在其中一扇后。你选门后主持人打开一扇空门，换门 vs 坚持，哪个策略胜率更高？编程验证。

```python
import random

def monty_hall(switch: bool, n: int = 100_000) -> float:
    wins = 0
    for _ in range(n):
        prize = random.randint(0, 2)
        choice = random.randint(0, 2)
        # Host opens a door ≠ choice and ≠ prize
        host = random.choice([d for d in range(3) if d != choice and d != prize])
        if switch:
            new_choice = next(d for d in range(3) if d != choice and d != host)
            wins += (new_choice == prize)
        else:
            wins += (choice == prize)
    return wins / n

print(f"Stay:   {monty_hall(switch=False):.3f}")  # ≈ 0.333
print(f"Switch: {monty_hall(switch=True):.3f}")   # ≈ 0.667
```

### P7: 接受-拒绝采样（从均匀分布生成任意分布）

**题目**：用 `Uniform(0,1)` 采样 `f(x) = 6x(1-x)`（Beta(2,2) 分布）。展示接受-拒绝采样通用框架。

**关键思路**：找包络 `M·g(x) ≥ f(x)`（这里 g=1, M=1.5），均匀生成候选点，以 `f(x)/(M·g(x))` 概率接受。

```python
import random

def rejection_sample_beta22(n: int = 10_000) -> list:
    """Sample from f(x)=6x(1-x) on [0,1] using rejection sampling"""
    M = 1.5  # max of f(x) = 6x(1-x) is 1.5 at x=0.5
    samples = []
    while len(samples) < n:
        x = random.random()
        u = random.random()
        if u <= 6 * x * (1 - x) / M:
            samples.append(x)
    return samples

s = rejection_sample_beta22()
print(f"Mean ≈ {sum(s)/len(s):.4f}")  # should be 0.5
print(f"Var  ≈ {sum((x-0.5)**2 for x in s)/len(s):.4f}")  # should be 0.05
```

### P8（原 Q16）：水库采样（Reservoir Sampling）

**题目**：实时数据流，维护大小为 k 的等概率随机样本，流中每个新元素仍满足等概率。

**关键思路**：第 i 个元素（i ≥ k）以概率 k/i 被选入，入选则随机替换现有样本之一。

```python
import random
from typing import Iterator

def reservoir_sample(stream: Iterator, k: int) -> list:
    """O(n) single-pass reservoir sampling"""
    reservoir = []
    for i, item in enumerate(stream):
        if i < k:
            reservoir.append(item)
        else:
            j = random.randint(0, i)     # inclusive
            if j < k:
                reservoir[j] = item
    return reservoir

print(reservoir_sample(iter(range(1, 101)), 5))  # 5 numbers from 1-100
```

### P9（原 Q22/Q46）：醉汉乘客问题

**题目**：100 座位 100 人，第一人随机坐，其余人自己座位被占则随机选。第 100 人坐到自己位子的概率？

**答案**：1/2（第 100 号座位与第 1 号座位被抢占的概率完全对称）

```python
import random

def airplane_seat(n: int = 100, trials: int = 100_000) -> float:
    successes = 0
    for _ in range(trials):
        seats = list(range(n))
        taken = [False] * n
        taken[random.randrange(n)] = True  # 1st passenger random

        for p in range(1, n - 1):
            if not taken[p]:
                taken[p] = True
            else:
                available = [s for s in range(n) if not taken[s]]
                taken[random.choice(available)] = True

        # Last passenger gets their seat (n-1) only if it's still free
        successes += not taken[n - 1]
    return successes / trials

print(f"P(last gets own seat) ≈ {airplane_seat():.3f}")  # ≈ 0.500
```

### P10（原 Q32）：有偏硬币 → 无偏（冯·诺伊曼方法）

**题目**：硬币以未知概率 p 出现正面，如何生成等概率 0/1？

**关键思路**：每次抛两枚，HT→1，TH→0（两者等概率 p(1-p)），否则重试。

```python
import random

def biased_coin(p: float) -> int:
    return 1 if random.random() < p else 0

def fair_from_biased(p: float) -> int:
    while True:
        a, b = biased_coin(p), biased_coin(p)
        if a != b:
            return a  # HT → 1, TH → 0

results = [fair_from_biased(0.3) for _ in range(100_000)]
print(f"P(1) ≈ {sum(results)/len(results):.4f}")  # ≈ 0.5000
# Expected trials per output: 1/(2p(1-p))，p=0.3时约 2.38 次抛硬币
```

**题目**：掷公平骰子，直到六个面都出现过至少一次，期望掷多少次？

**答案**：`E = 6(1 + 1/2 + 1/3 + 1/4 + 1/5 + 1/6) ≈ 14.70`

```python
import random

def coupon_collector(n: int = 6, trials: int = 100_000) -> float:
    total = 0
    for _ in range(trials):
        seen, rolls = set(), 0
        while len(seen) < n:
            seen.add(random.randint(1, n))
            rolls += 1
        total += rolls
    return total / trials

def theoretical(n: int) -> float:
    return n * sum(1 / k for k in range(1, n + 1))

print(f"Simulated:   {coupon_collector():.2f}")    # ≈ 14.70
```

```python
import random

def hh_vs_th(n: int = 100_000) -> float:
    """P(HH appears before TH)"""
    hh_wins = 0
    for _ in range(n):
        prev = random.choice('HT')
        while True:
            curr = random.choice('HT')
```

### P14（原 Q69）：圆内均匀采样

**题目**：如何在半径为 1 的圆内均匀随机选取一个点？

**常见错误**：直接用 `r = random()` 会导致偏向圆心（因面积元素为 r dr dθ，需 `r = sqrt(U)`）。

```python
import random, math

def uniform_in_circle_rejection():
    """Box rejection: O(1) expected calls, π/4 ≈ 78.5% acceptance rate"""
    while True:
        x, y = random.uniform(-1, 1), random.uniform(-1, 1)
        if x * x + y * y <= 1:
            return x, y

def uniform_in_circle_polar():
    """Polar method: r = sqrt(U) corrects for area element"""
    theta = random.uniform(0, 2 * math.pi)
    r = math.sqrt(random.random())   # NOT random.random() directly!
    return r * math.cos(theta), r * math.sin(theta)

# Verify: mean(r) should be 2/3
points = [uniform_in_circle_rejection() for _ in range(10_000)]
mean_r = sum(math.hypot(x, y) for x, y in points) / len(points)
print(f"Mean radius ≈ {mean_r:.3f}")  # should be ≈ 0.667
```

### P16（原 Q84）：Hash 碰撞期望（生日悖论推广）

**题目**：Hash 函数将对象等概率映射到 1-10。放入 10 个对象，(a) 发生碰撞的概率；(b) 碰撞次数的期望；(c) 空桶数的期望。

```python
import math

def hash_collision_stats(n: int = 10, k: int = 10):
    # (a) P(no collision) = k*(k-1)*...*(k-n+1) / k^n
    p_no_collision = 1.0
    for i in range(n):
        p_no_collision *= (k - i) / k
    p_collision = 1 - p_no_collision

    # (b) E[collision pairs] = C(n,2) / k
    e_collisions = math.comb(n, 2) / k

    # (c) E[empty buckets] = k * (1-1/k)^n
    e_empty = k * (1 - 1 / k) ** n

    return p_collision, e_collisions, e_empty

p, ec, ee = hash_collision_stats()
print(f"P(collision)     ≈ {p:.4f}")   # ≈ 0.6513
print(f"E[collisions]    = {ec:.4f}")  # = 4.5
print(f"E[empty buckets] ≈ {ee:.4f}")  # ≈ 3.487
```

### P17（原 Q90）：宝剑升级马尔可夫 DP ⭐

**题目**：宝剑升级：50% 升一级，50% 失败（≥5 级时失败降一级，<5 级失败原地不动）。从 1 级升到 9 级期望宝石数？

**关键思路**：

- 1-4 级：`E[k] = 1 + 0.5*E[k+1] + 0.5*E[k]` → `E[k] = 2 + E[k+1]`
- 5-8 级：`E[k] = 1 + 0.5*E[k+1] + 0.5*E[k-1]`（联立方程）

```python
import numpy as np

def sword_upgrade_expected() -> dict:
    """E[k] = expected gems to go from level k to level 9"""
    # Levels 5-8: substitute E[4] = 2 + E[5] into E[5]'s equation
    # E[5] = 1 + 0.5*E[6] + 0.5*(2+E[5])  =>  E[5] = 4 + E[6]
    # Similarly derive: E[6] = 6+E[7], E[7] = 8+E[8]
    # E[8] = 1 + 0.5*0 + 0.5*E[7]  =>  E[8] = 1 + 0.5*(8+E[8])  =>  E[8] = 10

    A = np.array([
        [ 1,  -1,   0,   0],   # E[5] - E[6] = 4
        [-0.5, 1, -0.5,  0],   # -0.5E[5] + E[6] - 0.5E[7] = 1
        [ 0, -0.5,  1, -0.5],  # -0.5E[6] + E[7] - 0.5E[8] = 1
        [ 0,   0, -0.5,  1],   # -0.5E[7] + E[8] = 1
    ])
    b = np.array([4., 1., 1., 1.])
    sol = np.linalg.solve(A, b)

    E = {9: 0}
    for i, k in enumerate([5, 6, 7, 8]):
        E[k] = sol[i]
    for k in range(4, 0, -1):
        E[k] = 2 + E[k + 1]
    return E

E = sword_upgrade_expected()
for k in range(1, 10):
    print(f"Level {k} → 9: {E[k]:.0f} gems")
# Level 1 → 9: 36 gems
# Level 8 → 9: 10 gems
```

### P19（原 Q97）：生日悖论模拟

**题目**：房间里需要多少人才能使至少两人同生日的概率 ≥ 50%？（精确计算 + 模拟两种实现）

```python
import random

def birthday_exact(n: int, days: int = 365) -> float:
    """P(collision) among n people"""
    p = 1.0
    for k in range(n):
        p *= (days - k) / days
    return 1 - p

def birthday_threshold(target: float = 0.5) -> int:
    n = 1
    while birthday_exact(n) < target:
        n += 1
    return n

print(f"Threshold: {birthday_threshold()} people")  # 23
print(f"P(23): {birthday_exact(23):.4f}")           # 0.5073

def birthday_sim(n: int, trials: int = 20_000) -> float:
    hits = sum(
        len(set(random.randint(1, 365) for _ in range(n))) < n
        for _ in range(trials)
    )
    return hits / trials

print(f"Sim P(23): {birthday_sim(23):.4f}")  # ≈ 0.507
```

### P20（原 Q98）：最优秘书问题（Secretary Problem）

**题目**：面试 n 个候选人（随机顺序），每次立即决定录用与否。最优策略？成功概率？

**答案**：先拒绝前 `⌊n/e⌋` 人，然后录用第一个超越所有已见候选人的人。成功概率 → 1/e ≈ 36.8%。

```python
import math, random

def secretary_problem(n: int, k: int = None, trials: int = 50_000) -> float:
    """Simulate optimal secretary strategy"""
    if k is None:
        k = max(1, int(n / math.e))
    successes = 0
    for _ in range(trials):
        candidates = list(range(n))
        random.shuffle(candidates)
        best_seen = max(candidates[:k])
        hired = next((c for c in candidates[k:] if c > best_seen), None)
        if hired == n - 1:  # best candidate overall
            successes += 1
    return successes / trials

n = 100
print(f"Optimal k = {int(n/math.e)}")  # 36
for k in [30, 36, 37, 38, 40]:
    print(f"k={k}: {secretary_problem(n, k):.3f}")  # peak ≈ 0.368 near k=36
```

### P21（原 Q102）：一维随机游走期望步数

**题目**：从位置 0 出发做一维对称随机游走（每步 ±1 等概率），首次到达位置 N 的期望步数？

**答案**：`E[T] = N²`（可选停止定理 / 鞅方法）

```python
import random

def random_walk_expected(N: int, trials: int = 20_000) -> float:
    total = 0
    for _ in range(trials):
        pos, steps = 0, 0
        while pos < N:
            pos += 1 if random.random() < 0.5 else -1
            steps += 1
        total += steps
    return total / trials

for N in [5, 10, 20]:
    sim = random_walk_expected(N)
    print(f"N={N}: sim={sim:.1f}, theory={N**2}")
```

### P22（原 Q114）：HHT 与 HTT 哪个先出现？

**题目**：连续抛公平硬币，HHT 和 HTT 哪个序列更快出现？HHT 先出现的概率？

**答案**：P(HHT 先) = 2/3（Markov 链方程：HHT 利用了 HH 的"前缀共享"）

```python
import random

def hht_vs_htt(trials: int = 100_000) -> float:
    """P(HHT appears before HTT) via simulation"""
    hht_wins = 0
    for _ in range(trials):
        history = []
        while True:
            history.append(random.choice('HT'))
            if len(history) >= 3:
                last = ''.join(history[-3:])
                if last == 'HHT':
                    hht_wins += 1
                    break
                elif last == 'HTT':
                    break
    return hht_wins / trials

print(f"P(HHT before HTT) ≈ {hht_vs_htt():.3f}")  # ≈ 0.667
```

### P23（原 Q118）：电梯停靠期望次数

**题目**：8 人随机独立选择 10 层楼之一，电梯期望停靠几次？

**答案**：`E = 10 × [1 - (9/10)⁸] ≈ 5.70`

```python
import random

def elevator_expected_exact(n: int = 8, k: int = 10) -> float:
    """E[stops] = k * (1 - ((k-1)/k)^n)"""
    return k * (1 - ((k - 1) / k) ** n)

def elevator_sim(n: int = 8, k: int = 10, trials: int = 100_000) -> float:
    return sum(
        len({random.randint(1, k) for _ in range(n)})
        for _ in range(trials)
    ) / trials

print(f"Exact:   {elevator_expected_exact():.4f}")   # 5.6953
print(f"Sim:     {elevator_sim():.4f}")              # ≈ 5.695
```
