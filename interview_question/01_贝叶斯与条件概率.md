> 包含：贝叶斯定理、条件概率、后验概率、悖论（两信封、蒙提霍尔）、辛普森悖论等面试题。

## Q5: 五个硬币有两个两面都是花，两个两面都是字，一个一花一字，问给定正面为字的情况下背面为字的概率

**答案：4/5**

用贝叶斯定理。看到正面是"字"，可能来自：
- 两个双字硬币：每个都100%显示字面，共贡献 2 个等权事件
- 一个一花一字硬币：50%概率显示字面，贡献 0.5 个等权事件

P(正面=字 | 双字) = 1，有2枚 → 权重 2×1 = 2
P(正面=字 | 正常) = 1/2，有1枚 → 权重 1×0.5 = 0.5

P(背面=字 | 正面=字) = P(双字硬币 | 正面=字) = 2 / (2 + 0.5) = 2/2.5 = **4/5 = 0.8**

## Q11: 共抛了5次，得到了两次正面，三次反面；下次一抛，是正面的概率是多少

**答案：1/2**

如果硬币是公平的，则每次抛掷都是独立事件，下一次正面概率仍为 **1/2**。

（贝叶斯角度：如果不确定硬币是否公平，使用 Beta 分布后验。先验 Beta(1,1)，观测2正3反后，后验为 Beta(3,4)，下次正面概率的期望为 3/7。但经典概率下答案为 1/2。）

## Q12: 共抛了50次，得到了二十次正面，三十次反面；下一次抛，是正面的概率是多少

**答案：经典概率角度 1/2；贝叶斯角度 21/52**

- **频率学派**：硬币无记忆，下一次仍为 1/2。
- **贝叶斯学派**：先验 Beta(1,1)，观测后后验 Beta(21,31)，预测下一次正面概率为 E = 21/52 ≈ 0.404。

面试时两种角度都应讨论。

## Q20: 三扇门问题（蒙提霍尔问题），A重新选择后选中豪车的概率

**答案：2/3**

经典蒙提霍尔问题。

- A 初始选对的概率 = 1/3，此时换门会输。
- A 初始选错的概率 = 2/3，此时主持人排除了另一个错误选项，换门一定赢。

所以换门后赢的概率 = **2/3**。

## Q27: 降水概率问题

已知条件：
- P(降水) = 0.2，P(不降水) = 0.8
- P(预测降水 | 实际降水) = 0.9
- P(预测降水 | 实际不降水) = 0.3（注意：题目说"预测不会降水实际也会降水0.3"，应理解为误报率）

**求P(实际降水 | 预测降水)：**

P(预测降水) = P(预测降水|降水)P(降水) + P(预测降水|不降水)P(不降水)
= 0.9 × 0.2 + 0.3 × 0.8 = 0.18 + 0.24 = 0.42

P(实际降水|预测降水) = P(预测降水|降水)P(降水) / P(预测降水) = 0.18/0.42 = **3/7 ≈ 42.86%**

**预测成功的概率：**

P(成功) = P(预测降水且降水) + P(预测不降水且不降水)
= 0.9×0.2 + 0.7×0.8 = 0.18 + 0.56 = **0.74 = 74%**

## Q41: 一千枚硬币有一枚两面全是花，其余999正常，拿出一个硬币抛十次全是花，问这枚硬币是不正常的概率

**答案：**

贝叶斯定理：

P(不正常 | 10次花) = P(10次花 | 不正常) × P(不正常) / P(10次花)

- P(不正常) = 1/1000
- P(10次花 | 不正常) = 1
- P(10次花 | 正常) = (1/2)^10 = 1/1024
- P(10次花) = 1 × 1/1000 + (1/1024) × 999/1000 = (1024 + 999) / (1000 × 1024) = 2023/1024000

P(不正常 | 10次花) = (1/1000) / (2023/1024000) = 1024/2023 ≈ **50.62%**

## Q53: 一个家庭有两个孩子，已知其中一个是女孩，求另一个也是女孩的概率

**答案：1/3**

样本空间（等概率）：{男男, 男女, 女男, 女女}

"至少一个女孩"排除了"男男"，剩下 {男女, 女男, 女女}。

P(两个都是女孩 | 至少一个女孩) = 1/3。

注意：如果条件是"**指定的那个**（如老大）是女孩"，则答案是 1/2。

## Q68: 1/1000 人患有某种疾病，检测准确率98%（有病检出阳性），假阳性率1%。某人检测阳性，实际患病的概率？

**答案：约 8.93%**

**贝叶斯定理的经典应用**：

- P(患病) = 1/1000 = 0.001
- P(阳性|患病) = 0.98
- P(阳性|未患病) = 0.01

$$P(\text{阳性}) = 0.98 \times 0.001 + 0.01 \times 0.999 = 0.00098 + 0.00999 = 0.01097$$

$$P(\text{患病}|\text{阳性}) = \frac{0.00098}{0.01097} \approx \textbf{8.93\%}$$

**面试重点**：虽然检测准确率高达98%，但因为疾病本身极为罕见（先验概率低），阳性结果中大部分是假阳性。这就是为什么罕见病需要**二次检测**确认。

**直觉**：1000人中1人真正患病（被正确检出），但999个健康人中约10人会假阳性。所以阳性人群中，真正患病的只有 1/(1+10) ≈ 9%。

## Q72: 三个朋友分别告诉你西雅图在下雨，每人有 1/3 概率说谎。假设任意一天下雨概率0.25，实际下雨的概率是多少？

**答案：8/11 ≈ 72.7%**

**贝叶斯定理 + 独立证据**：

设 R = 下雨，E = 三人都说下雨。

每个人：
- 下雨时说"下雨"概率 = 2/3（不说谎）
- 没下雨时说"下雨"概率 = 1/3（说谎）

三人独立：
- P(三人都说下雨 | R) = (2/3)³ = 8/27
- P(三人都说下雨 | ¬R) = (1/3)³ = 1/27

$$P(R|E) = \frac{P(E|R) \cdot P(R)}{P(E|R) \cdot P(R) + P(E|\neg R) \cdot P(\neg R)}$$

$$= \frac{\frac{8}{27} \times 0.25}{\frac{8}{27} \times 0.25 + \frac{1}{27} \times 0.75} = \frac{8 \times 0.25}{8 \times 0.25 + 1 \times 0.75} = \frac{2}{2 + 0.75} = \frac{2}{2.75} = \frac{8}{11} \approx \textbf{72.7\%}$$

## Q77: Facebook 内容审核问题 — 90%审核员是认真的（标记20%内容为垃圾），10%不认真（标记0%为垃圾）。一个审核员连续标记4条内容为"正常"，他是认真审核员的概率？

**答案：(0.9 × 0.8⁴) / (0.9 × 0.8⁴ + 0.1 × 1⁴) ≈ 78.7%**

**贝叶斯定理**：

设 D = 认真，¬D = 不认真。

- P(D) = 0.9, P(¬D) = 0.1
- P(4条正常 | D) = 0.8⁴ = 0.4096（每条80%概率标为正常）
- P(4条正常 | ¬D) = 1⁴ = 1（不认真的永远标正常）

$$P(D | \text{4条正常}) = \frac{0.9 \times 0.4096}{0.9 \times 0.4096 + 0.1 \times 1} = \frac{0.36864}{0.36864 + 0.1} = \frac{0.36864}{0.46864} \approx \textbf{78.7\%}$$

观察到更多"正常"标记后，P(认真) 会继续下降。N 条全标正常后：

$$P(D | N\text{条正常}) = \frac{0.9 \times 0.8^N}{0.9 \times 0.8^N + 0.1}$$

## Q85: 你有一个公平硬币和一个双面硬币（两面都是正面），随机选一个抛5次全部正面。选到双面硬币的概率？

**答案：32/33 ≈ 97.0%**

**贝叶斯定理**：

- P(双面) = P(公平) = 1/2
- P(5正 | 双面) = 1
- P(5正 | 公平) = (1/2)^5 = 1/32

$$P(\text{双面} | \text{5正}) = \frac{1 \times 1/2}{1 \times 1/2 + (1/32) \times 1/2} = \frac{1}{1 + 1/32} = \frac{32}{33} \approx \textbf{97.0\%}$$

**与 Q41 的对比**：Q41 中不正常硬币的先验很低（1/1000），需要10次全花才达到50%。本题先验高（1/2），5次就达到97%。**先验概率对后验影响巨大。**

## Q99: 两信封悖论（Two-Envelope Problem）

**题目**：两个信封，一个金额是另一个的 2 倍。随机拿起一个，看到金额 x，要不要换？

**悖论论证（错误）**：

另一个信封有 2x（概率 1/2）或 x/2（概率 1/2），换后期望：

$$E[\text{换后}] = \frac{1}{2}(2x) + \frac{1}{2}\cdot\frac{x}{2} = \frac{5x}{4} > x$$

这意味着无论 x 是多少都应该换——但由对称性换与不换结果应该相同，矛盾！

**悖论解析**：

设两信封金额为 m 和 2m。你拿到 x，则 x = m 或 x = 2m，两者非对称先验。

正确计算：A 信封有 m 的概率 1/2，有 2m 的概率 1/2：

$$E[\text{换后}] = \frac{1}{2}(2m) + \frac{1}{2}(m) = \frac{3m}{2} = E[\text{不换}]$$

**关键错误**：对同一观测值 x，不能同时赋予"另一个是 2x"和"另一个是 x/2"各 1/2 的概率——这取决于 x 与 m 的关系，并非对称。

**结论**：换与不换期望相同，均为 3m/2。

## Q115: 面试反馈问题 — 感觉良好但结果不确定，用贝叶斯估计进入第二轮的概率

**题目**：
- P(进入第二轮) = 50%
- P(感觉良好 \| 进入第二轮) = 95%
- P(感觉良好 \| 未进入第二轮) = 75%

**你感觉良好，进入第二轮的概率是多少？**

**答案：≈ 55.9%**

**贝叶斯公式**：

$$P(\text{感觉良好}) = 0.95 \times 0.5 + 0.75 \times 0.5 = 0.475 + 0.375 = 0.85$$

$$P(\text{第二轮} \mid \text{感觉良好}) = \frac{0.95 \times 0.5}{0.85} = \frac{0.475}{0.85} = \frac{19}{34} \approx 55.9\%$$

**直觉为何只有55.9%而非95%**：感觉良好时，75%的未晋级者也感觉良好（false positive 率很高），因此大量未晋级者"稀释"了先验置信度。

> **面试考点**：此题考察对"基率（Base Rate）"的重视程度。即使 P(感觉好|第二轮)=0.95 很高，如果base rate只有50%，结论仍然不强。这是数据科学中**精确率-召回率权衡**的直觉来源。

## Q125: 辛普森悖论（Simpson's Paradox）

**题目**：两家医院 A 和 B，数据如下。总体看 A 的存活率低于 B，但在轻症和重症中 A 都高于 B。这可能吗？请解释原因。

| | 医院 A | | 医院 B | |
|---|---|---|---|---|
| 病情 | 存活/总数 | 存活率 | 存活/总数 | 存活率 |
| 轻症 | 81/90 | **90%** | 234/270 | 87% |
| 重症 | 192/210 | **73%** | 55/80 | 69% |
| **总体** | 273/300 | **91%** | 289/350 | **83%** |

等等——A 总体存活率 91% > B 的 83%？重新算一下：

| | 医院 A | | 医院 B | |
|---|---|---|---|---|
| 轻症 | 81/90 | **90%** | 234/270 | 87% |
| 重症 | 192/210 | **73%** | 55/80 | 69% |
| **总体** | 273/300 | **91%** | 289/350 | **83%** |

实际上在这个数字设定里 A 总体更好。让我们用一个"悖论方向"的例子：

| | 医院 A | | 医院 B | |
|---|---|---|---|---|
| 轻症 | 18/20 | **90%** | 216/240 | **90%**（同） |
| 重症 | 2/100 | **2%** | 7/80 | **8.75%**（B更好）|
| **总体** | 20/120 | 16.7% | 223/320 | **69.7%（B反而总体更好）** |

更典型的悖论案例（伯克利大学录取问题）：

| | 男性申请 | | 女性申请 | |
|---|---|---|---|---|
| 部门A | 825/1000 | **82.5%** | 108/120 | 90%（女更高）|
| 部门B | 560/4000 | 14% | 25/600 | **4.2%**（男更高？）|
| **总体** | 1385/5000 | **27.7%** | 133/720 | 18.5% |

**表面看**：总体男性录取率（27.7%）远高于女性（18.5%）→ 疑似歧视女性。
**实际**：女性倾向于申请竞争更激烈的部门（B），而男性更多申请竞争少的部门（A）。在每个部门内，并没有对女性的歧视（甚至部门A女性录取率更高）。

**Simpson's Paradox 的本质**：

当存在一个**混淆变量（Confounding Variable）**时，汇总数据中的趋势可能与分层数据中的趋势**完全相反**。

混淆变量扭曲了结果：
- 医院案例：病情严重程度是混淆变量（A 接诊更多重症患者）
- 大学案例：部门选择是混淆变量（女性选择了更难录取的部门）

**数据科学中的启示**：
1. **永远要做分层分析（Stratified Analysis）**，不能只看汇总数字
2. A/B 测试若忘记控制分群（如移动端 vs 桌面端），可能出现辛普森悖论
3. 因果推断中，Confounder 不控制会导致完全错误的结论
4. 解决方案：**随机化实验（RCT）** 是最干净的方式；观察研究需用倾向评分匹配（PSM）或分层回归

> **Follow-up 提示：** 如何判断哪个层面的数字才是"正确的"答案？这取决于因果关系的方向。若部门选择是女性**主动决策**的结果（非歧视导致的），则分部门看才是正确视角。若部门选择本身就是歧视的结果（女性被迫只能选难部门），则总体视角才对。辛普森悖论无法从数据本身解决，需要领域知识和因果图（Causal Graph/DAG）。

## Q128: 条件独立与朴素贝叶斯（Naive Bayes 的概率假设）

**题目**：给定类别标签 C，若特征 X₁ 和 X₂ 在给定 C 后条件独立，是否意味着 X₁ 和 X₂ 无条件也独立？

**答案：不一定。条件独立 ≠ 无条件独立。**

**定义**：
- **条件独立**：P(X₁, X₂ | C) = P(X₁|C) · P(X₂|C)
- **无条件独立**：P(X₁, X₂) = P(X₁) · P(X₂)

**反例**：

考虑硬币问题：C = 选择公平硬币（p=0.5）或双面正面硬币（p=1），各以 1/2 概率选择。
- X₁ = 第一次抛出正面，X₂ = 第二次抛出正面

给定 C（知道选了哪个硬币），X₁ 和 X₂ 条件独立（两次抛掷独立）。

但是边缘上：P(X₂ = H | X₁ = H) > P(X₂ = H)，因为 X₁ = H 增加了是双面硬币的概率。

所以 X₁ 和 X₂ **无条件不独立**。

**Naive Bayes 的条件独立假设**：

NB 假设：给定类别 C，所有特征条件独立：
$$P(X_1, X_2, \ldots, X_n | C) = \prod_{i=1}^n P(X_i | C)$$

- 这是**极强的假设**，现实中往往不满足（词语共现、像素相关等）
- 即使假设不满足，NB 在分类任务中往往依然表现良好（决策边界的预测通常足够准确）
- 当特征确实高度相关时，NB 会不适当地重复计数某些证据，需用 LDA、Logistic Regression 等模型替代

> **Follow-up 提示：** Naive Bayes 与 Logistic Regression 的关系？可以证明 NB 的对数概率比（log-odds）是特征的线性函数，与 Logistic Regression 的决策边界结构相同。NB 是**生成模型**（建模联合分布），LR 是**判别模型**（直接建模条件分布）。
