## Q3: 在A地有两辆公交车，一辆间隔5分钟，一辆间隔7分钟，问等车时间的期望。公交站每一分钟出现公交车的概率是5%，15分钟之后仍然没有公交车的概率是多少

**第一问：等车时间的期望**

假设两辆公交车的到达时间相互独立且均匀分布。

- 等第一辆车（间隔5分钟）的期望等待时间 = 5/2 = 2.5分钟
- 等第二辆车（间隔7分钟）的期望等待时间 = 7/2 = 3.5分钟

等任意一辆车的等待时间 T = min(T₁, T₂)。

T₁ ~ Uniform(0,5)，T₂ ~ Uniform(0,7)。

P(T > t) = P(T₁ > t) · P(T₂ > t)

对于 t ≤ 5：P(T > t) = (5-t)/5 · (7-t)/7 = (5-t)(7-t)/35

E[T] = ∫₀⁵ P(T > t) dt = ∫₀⁵ (5-t)(7-t)/35 dt = ∫₀⁵ (35 - 12t + t²)/35 dt

= (1/35)[35t - 6t² + t³/3]₀⁵ = (1/35)(175 - 150 + 125/3) = (1/35)(25 + 125/3) = (1/35)(200/3) = **40/21 ≈ 1.905分钟**

**第二问：每分钟出现公交车的概率5%，15分钟后仍然没有公交车的概率**

P(15分钟无车) = (1 - 0.05)^15 = 0.95^15 ≈ **0.4633 ≈ 46.33%**

## Q6: 甲和乙两人投硬币，甲先投，先投出正面朝上的获得胜利，求甲胜利的概率

**答案：2/3**

设甲获胜概率为 P。
- 甲第一次投正面（概率1/2），甲赢。
- 甲第一次反面（概率1/2），轮到乙。乙正面（概率1/2），乙赢。乙反面（概率1/2），回到起点。

P = 1/2 + (1/2)(1/2)P = 1/2 + P/4

3P/4 = 1/2 → P = **2/3**

## Q8: 出现连续一正一反和连续两个正面即停止，哪个期望值大

**答案：出现连续两个正面的期望次数更大。**

**连续"正反"(HT) 的期望：**

用**马尔可夫链**建模，定义状态和转移：

| 状态 | 含义 | 抛H（概率1/2） | 抛T（概率1/2） |
|------|------|----------------|----------------|
| S₀ | 初始/上次是T | → S_H | → S₀（T不是H，重来）|
| S_H | 上次是H | → S_H（HH，还没出T）| → **结束**（HT达成！）|

列方程（E = 还需要多少次抛掷）：
- E₀ = 1 + (1/2)E_H + (1/2)E₀
- E_H = 1 + (1/2)E_H + (1/2)×0

由 E_H：E_H - E_H/2 = 1 → **E_H = 2**
代入 E₀：E₀ = 1 + 1 + E₀/2 → E₀/2 = 2 → **E₀ = 4**

**连续"正正"(HH) 的期望：**

| 状态 | 含义 | 抛H（概率1/2） | 抛T（概率1/2） |
|------|------|----------------|----------------|
| S₀ | 初始/上次是T | → S_H | → S₀ |
| S_H | 上次是H | → **结束**（HH达成！）| → S₀（**前功尽弃！**）|

列方程：
- E₀ = 1 + (1/2)E_H + (1/2)E₀
- E_H = 1 + (1/2)×0 + (1/2)E₀

由 E_H = 1 + E₀/2，代入 E₀：
E₀ = 1 + (1/2)(1 + E₀/2) + E₀/2 = 3/2 + 3E₀/4
E₀/4 = 3/2 → **E₀ = 6**

**结论**：HH 期望 **6 次** > HT 期望 **4 次**。

**直觉解释**：两者的关键差别在于"失败后的代价"：
- 追求 HT 时，在 S_H 状态抛到 H → 停留在 S_H（没有退步，下次抛 T 仍可结束）
- 追求 HH 时，在 S_H 状态抛到 T → 回到 S₀（完全重来！）

HH 的"失败惩罚"更重，所以期望更大。

## Q13: 求几何分布的期望

**答案：E[X] = 1/p**

X ~ Geo(p)，P(X=k) = (1-p)^(k-1) · p，k = 1, 2, 3, ...

$$E[X] = \sum_{k=1}^{\infty} k(1-p)^{k-1}p = p \cdot \frac{1}{(1-(1-p))^2} = p \cdot \frac{1}{p^2} = \frac{1}{p}$$

## Q28: a 个球放到 b 个篮子，平均多少个篮子有球

**答案：b · [1 - (1 - 1/b)^a]**

每个篮子为空的概率 = (1 - 1/b)^a

每个篮子有球的概率 = 1 - (1 - 1/b)^a

期望有球的篮子数 = **b · [1 - (1-1/b)^a]**

当 a, b 都很大时，近似为 b(1 - e^(-a/b))。

## Q30: 甲乙两人玩游戏，每回合双方胜率都是50%，甲累计赢两场就算获胜，乙累计赢三场就获胜，问甲获胜的概率

**答案：11/16**

甲需赢2场，乙需赢3场。最多进行 2+3-1=4 场比赛。

枚举所有情况（甲用A表示，乙用B表示）：

甲在恰好第 k 场获胜 = 前 k-1 场甲赢了1场，第 k 场甲赢。

- 甲2场赢：AA → 概率 1/4
- 甲3场赢：ABA, BAA → 概率 2/8 = 1/4
- 甲4场赢：前3场甲赢1场乙赢2场，第4场甲赢：ABBA, BABA, BBAA → 概率 3/16

P(甲赢) = 1/4 + 1/4 + 3/16 = 4/16 + 4/16 + 3/16 = **11/16 = 68.75%**

## Q38: 甲乙两人比赛，赢的概率分别为 p 和 1-p，假定连续两次赢才算获胜，那最后甲获胜的概率为多大

**答案：**

设状态：
- S₀：初始状态（或上一轮无人连胜）
- S_A：甲刚赢一局
- S_B：乙刚赢一局

令 q = 1-p。从各状态出发甲最终获胜的概率：

- P₀ = p·P_A + q·P_B
- P_A = p·1 + q·P_B（甲再赢就获胜）
- P_B = p·P_A + q·0（乙再赢则甲输）

从 P_B = p·P_A，得 P_A = P_B/p。
代入 P_A = p + q·P_B：P_B/p = p + q·P_B → P_B = p² + pq·P_B → P_B(1-pq) = p² → P_B = p²/(1-pq)

P_A = P_B/p = p/(1-pq)

P₀ = p·P_A + q·P_B = p²/(1-pq) + qp²/(1-pq) = p²(1+q)/(1-pq)

$$P(\text{甲获胜}) = \frac{p^2(1+q)}{1-pq} = \frac{p^2(2-p)}{1-p(1-p)} = \frac{p^2(2-p)}{1-p+p^2}$$

当 p = 1/2 时：P = (1/4)(3/2) / (3/4) = (3/8)/(3/4) = 1/2 ✓

## Q40: 一辆车经过p点会有50%的概率变速，速度可以是一分钟/圈或两分钟/圈。若随机时间到达p点等车，求等车时间的期望

**答案：**

在稳态下，车以两种速度运行：
- 1分钟/圈的概率 → 比例为 1/2
- 2分钟/圈的概率 → 比例为 1/2

但因为速度慢的圈占用更多时间，随机到达时遇到慢圈的概率更高。

稳态下，一圈1分钟和一圈2分钟各占50%的**圈数**，但按**时间**加权：
- 1分钟圈占时间比例：1/(1+2) = 1/3
- 2分钟圈占时间比例：2/(1+2) = 2/3

等车时间 = 所在圈剩余时间的期望：
- 在1分钟圈内：期望等待 1/2 分钟
- 在2分钟圈内：期望等待 2/2 = 1 分钟

E[等待] = (1/3)(1/2) + (2/3)(1) = 1/6 + 2/3 = **5/6 分钟 ≈ 50秒**

## Q42: 手枪每次50%概率发射子弹，甲乙轮流射击，甲先射击，乙先中弹的概率

直觉：先手有优势，因为先手有第一次射击的机会。

## Q45: 一个虫子，一个周期内，死掉概率1/3，活下去并繁殖两个后代概率2/3，求不会灭绝的概率

**答案：1/2**

**建模**：这是**分支过程（Branching Process）**问题。设 q 为从一只虫子出发最终灭绝的概率。

每只虫子独立：
- 概率 1/3 → 死亡（0个后代）
- 概率 2/3 → 产生2个后代，两个后代各自独立灭绝概率也是 q

**列方程**：灭绝 = 死亡 ∪ 两个后代都灭绝

$$q = \frac{1}{3} + \frac{2}{3}q^2$$

化简：3q = 1 + 2q² → **2q² - 3q + 1 = 0** → (2q-1)(q-1) = 0

解：q = 1/2 或 q = 1

**如何选择解？** 分支过程理论告诉我们：
- 平均后代数 μ = 0 × (1/3) + 2 × (2/3) = **4/3 > 1**（超临界过程）
- 当 μ > 1 时，灭绝概率是方程的**最小非负解**
- 当 μ ≤ 1 时，灭绝概率 = 1（必然灭绝）

这里 μ = 4/3 > 1，所以取最小解 q = 1/2。

**不灭绝概率 = 1 - 1/2 = 1/2**

**直觉理解**：虫子平均每代后代数4/3 > 1，种群有增长趋势。但由于随机波动（有1/3概率直接死亡），仍然有一半的可能灭绝。

## Q49: 公交车平均十分钟一辆，某个人在公交站点等待时间的期望是多少

**答案：取决于到达过程的分布，5分钟或10分钟**

**情况一：公交车严格等间隔10分钟**
- 人随机到达，在周期内均匀分布
- 等待时间 ~ U(0, 10)，期望 = **5分钟**

**情况二：公交车按泊松过程到达（平均10分钟一辆）**
- 到达间隔 X ~ Exp(λ=1/10)
- 由**无记忆性**：到达后的剩余等待时间仍服从 Exp(1/10)
- 期望等待时间 = 1/λ = **10分钟**

**为什么泊松过程等这么久（检验悖论 / Inspection Paradox）？**

直觉上"平均10分钟"应该等5分钟，但泊松过程的间隔有长有短（方差大）。随机到达的人**更可能**落在一个较长的间隔里（长间隔"捕获"随机到达者的概率更大）。

数学上：随机到达者所在间隔的长度不是 E[X]=10，而是按长度加权的 E[X²]/E[X]。对指数分布 E[X²] = 2/λ²，所以所在间隔期望长度 = 2/λ² ÷ 1/λ = 2/λ = 20分钟。在20分钟的间隔中随机落点，期望等待 = 10分钟。

**面试中两种情况都应讨论**，并解释为什么结果不同。

## Q50: 电池运行概率问题

已知：
- P(再运行2年 | 已运行3年) = 0.4 → P(运行≥5年 | 运行≥3年) = 0.4
- P(再运行2年 | 已运行4年) = 0.5 → P(运行≥6年 | 运行≥4年) = 0.5
- P(再运行3年 | 已运行3年) = 0.3 → P(运行≥6年 | 运行≥3年) = 0.3

求 P(再运行1年 | 已运行4年) = P(运行≥5年 | 运行≥4年)

由条件概率：
P(运行≥5年 | 运行≥3年) = P(运行≥5年 ∩ 运行≥3年) / P(运行≥3年) = P(≥5) / P(≥3) = 0.4

P(运行≥6年 | 运行≥3年) = P(≥6) / P(≥3) = 0.3

P(运行≥6年 | 运行≥4年) = P(≥6) / P(≥4) = 0.5

从上面的等式：
- P(≥5) = 0.4 × P(≥3)
- P(≥6) = 0.3 × P(≥3)
- P(≥6) = 0.5 × P(≥4)

所以 0.3 × P(≥3) = 0.5 × P(≥4) → P(≥4) = 0.6 × P(≥3)

P(≥5 | ≥4) = P(≥5) / P(≥4) = 0.4 × P(≥3) / (0.6 × P(≥3)) = 0.4/0.6 = **2/3 ≈ 66.67%**

## Q51: 抛一个骰子，直到集齐六面，问抛骰子的期望次数

**答案：14.7次**

这是**优惠券收集问题（Coupon Collector's Problem）**。

收集到第 i 面后，要收集第 i+1 面的概率为 (6-i)/6，期望次数为 6/(6-i)。

$$E = \sum_{i=0}^{5} \frac{6}{6-i} = \frac{6}{6} + \frac{6}{5} + \frac{6}{4} + \frac{6}{3} + \frac{6}{2} + \frac{6}{1}$$

$$= 1 + 1.2 + 1.5 + 2 + 3 + 6 = \textbf{14.7}$$

## Q52: 抛色子连续 n 次正面向上的期望次数

> 注："色子"即骰子。题目说"正面向上"对骰子不太准确，这里讨论两种理解。

**通用公式推导**：设每次成功概率为 p，求首次出现连续 n 次成功的期望次数 Fₙ。

**递推关系**：先花 Fₙ₋₁ 次达到连续 n-1 次成功，再抛一次：
- 成功（概率 p）→ 连续 n 次，完成
- 失败（概率 1-p）→ 前功尽弃，从头开始

Fₙ = Fₙ₋₁ + 1 + (1-p)·Fₙ → p·Fₙ = Fₙ₋₁ + 1 → **Fₙ = (Fₙ₋₁ + 1) / p**

**情况一：硬币（p = 1/2）**

Fₙ = 2Fₙ₋₁ + 2

| n | Fₙ |
|---|---|
| 1 | 2 |
| 2 | 6 |
| 3 | 14 |
| 4 | 30 |

通项公式：**Fₙ = 2^(n+1) - 2**

**情况二：骰子指定某一面（p = 1/6）**

Fₙ = 6Fₙ₋₁ + 6

| n | Fₙ |
|---|---|
| 1 | 6 |
| 2 | 42 |
| 3 | 258 |

通项公式：**Fₙ = 6(6ⁿ - 1) / 5**

**一般通项**：Fₙ = (1/p)·((1/p)ⁿ - 1) / (1/p - 1) = ((1/p)^(n+1) - (1/p)) / (1/p - 1)

## Q54: 抽蓝球红球，蓝结束红放回继续，平均结束游戏抽取次数

**答案：取决于蓝红球数量**

设有 b 个蓝球和 r 个红球，共 n = b+r 个球。

每次抽球：
- 抽到蓝球（概率 b/n）→ 游戏结束
- 抽到红球（概率 r/n）→ 放回继续

这是几何分布，成功概率 p = b/n。

E[抽取次数] = **n/b = (b+r)/b**

若 b = r：E = 2

## Q63: 两盒火柴每盒n根，每次随机从任意盒取出一根，求当一个盒子被取完时，另一个盒子刚好还剩r根的概率

**答案：Banach 火柴盒问题**

$$P(r) = \binom{2n-r}{n} \cdot \left(\frac{1}{2}\right)^{2n-r}$$

**详细推导**：

"发现盒子空"意味着：某次去取火柴时，选到了已经空的盒子。设盒 A 先被取空。

**事件分解**（以盒 A 先空为例）：
- 前若干次取用中，恰好从 A 取了 n 根（取完），从 B 取了 n-r 根（剩余 r 根）
- 最后一次操作：选到盒 A，发现空了（此次不算有效取用）

设总操作次数 = (n + n-r) 有效取用 + 1 次发现空 = **2n - r + 1** 次

**概率计算**（盒 A 先空）：
- 第 2n-r+1 次选盒 A（概率 1/2）
- 前 2n-r 次中恰好 n 次选 A、n-r 次选 B
- 关键约束：第 n 次选 A 后（A 变空），之后不能再选到 A（否则游戏提前结束）
  - 即：第 n 次选 A 必须在前 2n-r 次的最后出现

**用分步计数处理约束**：设第 n 次选 A 在第 k 步（n ≤ k ≤ 2n-r）：
- 前 k-1 步中选 n-1 次 A：C(k-1, n-1) × (1/2)^(k-1)
- 第 k 步选 A：1/2
- 第 k+1 到 2n-r 步全选 B：(1/2)^(2n-r-k)
- 第 2n-r+1 步选 A（发现空）：1/2

对所有合法的 k 求和并应用 Hockey-stick 恒等式：

$$\sum_{k=n}^{2n-r} \binom{k-1}{n-1} = \binom{2n-r}{n}$$

所以 P(盒 A 先空，B 剩 r) = C(2n-r, n) × (1/2)^(2n-r+1)

**乘2**（盒 A 或盒 B 都可能先空，对称）：

$$P(r) = 2 \cdot \binom{2n-r}{n} \cdot \left(\frac{1}{2}\right)^{2n-r+1} = \binom{2n-r}{n} \cdot \left(\frac{1}{2}\right)^{2n-r}$$

**验证 r=0**：P(0) = C(2n, n) × (1/2)^(2n)，即两盒同时用完的概率。当 n 很大时，由 Stirling 近似 ≈ 1/√(πn) → 0，符合直觉。

## Q66: HH vs TH 博弈 — 你和朋友轮流抛硬币，出现 HH 你赢，出现 TH 朋友赢。你赢的概率是多少？

**答案：你赢的概率 = 1/4**

**状态分析**：

| 状态 | 含义 | 抛H（1/2） | 抛T（1/2） |
|------|------|-----------|-----------|
| S₀ | 初始状态 | → S_H | → S_T |
| S_H | 上一次是H | → **你赢(HH)** | → **朋友赢(TH，是HT？)** |
| S_T | 上一次是T | → S_H | → S_T |

等等，需要仔细理解题意。题目说序列中首次出现 HH 你赢，首次出现 TH 朋友赢。

关键观察：一旦出现 T，下一次出现 H 就是 TH → 朋友赢。所以你赢的唯一机会是**前两次就是 HH**。

因为：
- 如果第一次是 T → 进入 S_T 状态，之后只要出现 H 就是 TH（朋友赢），出现 T 则停留在 S_T
- 如果第一次是 H → 第二次 H 则 HH（你赢），第二次 T 则 TH（朋友赢，不对，HT不是TH...）

重新分析：我们看的是**连续两次**的模式。

| 状态 | 抛H（1/2） | 抛T（1/2） |
|------|-----------|-----------|
| S₀（起始） | → S_H | → S_T |
| S_H（上次H） | → **HH，你赢！** | → S_T（上次T）|
| S_T（上次T） | → **TH，朋友赢！** | → S_T |

设 P₀ = 从 S₀ 出发你赢的概率，P_T = 从 S_T 出发你赢的概率。

- P_T = (1/2)·0 + (1/2)·P_T = P_T/2 → P_T(1-1/2) = 0 → **P_T = 0**

一旦进入 S_T 状态，你永远无法赢（因为下一个 H 就是 TH，朋友赢）。

- P₀ = (1/2)·(1/2·1 + 1/2·P_T) + (1/2)·P_T = (1/2)(1/2) + 0 = **1/4**

**你赢的概率只有 1/4。这个游戏对你极度不公平。**

**深层原因**：TH 比 HH 更容易出现，因为 HH 需要连续两个 H，而 TH 只需要在任意一个 T 后面跟一个 H。这种"非传递性"是 Penney's Game 的核心思想。

## Q70: 桶中有 M 个白球，每分钟随机取一个涂成红色再放回，问将全部球涂红的期望时间

**答案：M × H(M) = M × (1 + 1/2 + 1/3 + ... + 1/M)**

这是**优惠券收集问题（Coupon Collector）**的变体。

当已有 k 个红球时，下次涂到新球（白球）的概率 = (M-k)/M。

涂红第 k+1 个球的期望等待 = M/(M-k)。

$$E = \sum_{k=0}^{M-1} \frac{M}{M-k} = M\sum_{j=1}^{M}\frac{1}{j} = M \cdot H(M)$$

例如 M = 10：E = 10 × (1 + 1/2 + ... + 1/10) ≈ 10 × 2.93 ≈ **29.3分钟**

当 M 很大时，H(M) ≈ ln(M) + γ（γ ≈ 0.5772 为欧拉常数），所以 E ≈ M(ln M + 0.5772)。

## Q73: 从标准52张扑克牌中依次抽牌，抽到第一张 A 时期望已经抽了多少张牌？

**答案：53/5 = 10.6 张**

**方法（对称性论证）**：

将52张牌排成一列。4张A把其余48张非A牌分成5个"间隔"（包括两端）。

由对称性，48张非A牌均匀分布在5个间隔中，第一个间隔（在第一张A之前）的期望长度 = 48/5。

第一张A之前有 48/5 张非A牌，加上第一张A本身：

$$E = \frac{48}{5} + 1 = \frac{53}{5} = \textbf{10.6张}$$

**另一种理解**：5个间隔各有期望 48/5 = 9.6 张非A牌。这是因为48张牌随机分配到5个位置，每个位置期望相等。

## Q74: 一个国家规定每对夫妻生到第一个男孩就停止，问最终男女比例是多少？

**答案：1:1**

**直觉上似乎女孩会多**（因为有些家庭会有很多女孩才生到男孩），但实际上：

每次生育都是独立事件，P(男) = P(女) = 1/2。

**无论停止规则是什么**，每次生育的性别概率不变。停止规则不影响已出生婴儿的性别分布。

**严格证明**：设有 N 对夫妻。

- 每对夫妻生 1 个男孩（确定）
- 每对夫妻生女孩数 X ~ Geo(1/2) - 1（0, 1, 2, ...），E[X] = 1

期望男孩总数 = N，期望女孩总数 = N × 1 = N。

**男女比例 = 1:1**

> **面试陷阱**：很多人直觉会说女孩多。关键洞察是停止规则只决定样本量，不改变每次试验的概率。

## Q84: Hash 函数将对象映射到 1-10（等概率），10个对象，求：(a) 发生碰撞的概率 (b) 碰撞次数的期望 (c) 空桶数的期望

**答案：**

**(a) P(至少一次碰撞)**：

P(无碰撞) = 10!/10^10 × 只取前10个 = 10 × 9 × 8 × ... × 1 / 10^10 = 10!/10^10

$$P(\text{无碰撞}) = \frac{10!}{10^{10}} = \frac{3628800}{10000000000} \approx 0.000363$$

P(碰撞) = 1 - 10!/10^10 ≈ **99.96%**

**(b) 碰撞次数的期望**：

定义碰撞 = 一对 (i,j) 映射到同一个桶。共 C(10,2) = 45 对。

每对碰撞的概率 = 1/10。

E[碰撞对数] = 45 × 1/10 = **4.5 对**

**(c) 空桶数的期望**：

每个桶为空的概率 = (9/10)^10 ≈ 0.3487

E[空桶数] = 10 × (9/10)^10 ≈ **3.49 个**

（与 Q28 相同公式：b·(1-1/b)^a，这里 a=b=10）

## Q90: 一个宝剑升级系统：使用宝石50%概率升一级，50%失败。等级 ≥ 5 时失败会降一级，< 5 时失败无效果。从1级升到9级的期望宝石数？

**答案：期望 = 12 + 2 × (1 + 3 + 7 + 15) = 64 个宝石**

**分两阶段分析**：

**阶段一（1→5级）：失败无惩罚**

从 k 级到 k+1 级：成功概率1/2，失败留在原地。这是几何分布，期望 = 2。

E[1→5] = 4 × 2 = **8**

**阶段二（5→9级）：失败降一级（赌徒漫步）**

从等级 k（k ≥ 5）升到 k+1：成功(+1)概率1/2，失败(-1)概率1/2。

这是有吸收壁的随机游走。设 E_k 为从等级 k 到等级 9 的期望步数，等级 4 是反射壁（降到4时等同于从4开始，但4→5无惩罚，期望2步回到5）。

更准确地建模：设 fₖ = 从等级 k 到等级 k+1 的期望步数（k ≥ 5）。

fₖ = 1 + (1/2)·0 + (1/2)·(fₖ₋₁ + fₖ) （失败后先回到 k-1，再回到 k，再继续）

fₖ = 1 + fₖ₋₁/2 + fₖ/2 → fₖ/2 = 1 + fₖ₋₁/2 → **fₖ = 2 + fₖ₋₁**

但 f₄ = 2（从4到5无惩罚）。

- f₅ = 2 + f₄ = 2 + 2 = 4

等等，这里的递推需要更仔细。从等级 k ≥ 5 尝试升级：
- 成功（1/2）→ 到 k+1，花1个宝石
- 失败（1/2）→ 降到 k-1，花1个宝石，然后需要先回到 k 再继续

设 gₖ = 从 k 到 k+1 的期望（k ≥ 5），其中从 k-1 回到 k 的期望也是 gₖ₋₁（相当于从 k-1 到 k）。

但 k-1 = 4 时（即 k = 5），从4升到5不受惩罚，期望2步。

gₖ = 1 + (1/2)·0 + (1/2)·(gₖ₋₁ + gₖ) （失败后花 gₖ₋₁ 步回到 k，再花 gₖ 步继续）

gₖ - gₖ/2 = 1 + gₖ₋₁/2 → gₖ = 2 + gₖ₋₁

- g₅ = 2 + g₄ = 2 + 2 = 4
- g₆ = 2 + g₅ = 2 + 4 = 6
- g₇ = 2 + g₆ = 2 + 6 = 8
- g₈ = 2 + g₇ = 2 + 8 = 10

E[5→9] = g₅ + g₆ + g₇ + g₈ = 4 + 6 + 8 + 10 = **28**

**总期望 = E[1→5] + E[5→9] = 8 + 28 = 36 个宝石**

## Q92: 互相选择问题 — A组和B组各 n 个人，每人从对方组随机选一个"最好朋友"。没有任何互选配对的概率？

**答案：((n-1)/n)^n × 适当修正 → 当 n 大时趋近 1/e ≈ 36.8%**

设 A 组第 i 人选了 B 组的 f(i)，B 组第 j 人选了 A 组的 g(j)。

互选配对 = 存在 (i,j) 使得 f(i) = j 且 g(j) = i。

**近似分析**：定义指示变量 Xᵢⱼ = 1 当 (i,j) 互选。

P(Xᵢⱼ = 1) = (1/n) × (1/n) = 1/n²

互选对数的期望 = n² × 1/n² = **1**

由于各 Xᵢⱼ 近似独立（n大时），互选对数近似泊松分布 Poi(1)。

P(无互选) ≈ e^(-1) ≈ **36.8%**

**精确公式**（n小时）：

每个 A 组成员独立选 B 组一人（概率各 1/n），每个 B 组成员独立选 A 组一人。

P(无互选) = [1 - 1/n]^n × 修正项

当 n = 1 时：P = 0（必然互选）。
当 n → ∞ 时：P → 1/e。

## Q93: 在 [0,1] 上随机取 n 个数，求最小值的期望

**答案：1/(n+1)**

与 Q31（最大值期望 n/(n+1)）对称。

设 m = min(X₁, ..., Xₙ)，P(m > x) = (1-x)^n。

$$E[m] = \int_0^1 P(m > x) dx = \int_0^1 (1-x)^n dx = \frac{1}{n+1}$$

**一般结论**：n 个 U(0,1) 的第 k 个顺序统计量期望 = k/(n+1)（与 Q34 一致）。

最小值 = 第1个 = 1/(n+1)，最大值 = 第n个 = n/(n+1)。

## Q101: 52 张扑克牌不放回抽取，抽到第一张 A 的期望次数是多少

**答案：E = 53/5 = 10.6 次**

52 张牌中有 4 张 A，48 张非 A。

**推导（对称性 / 间隔法）**：

4 张 A 将整副牌的随机排列自然划分为 5 段（首段 + 4 张 A 之间的 3 段 + 尾段）。

由对称性，48 张非 A 均匀分布在 5 段中，每段期望含 48/5 张非 A。

$$E[\text{第一张 A 的位置}] = \frac{48}{5} + 1 = \frac{53}{5} = 10.6$$

**一般结论**：n 张牌中有 k 张特殊牌，第一张特殊牌的期望位置 = **(n+1)/(k+1)**

对本题：(52+1)/(4+1) = 53/5 = 10.6 ✓

> **对比 Q44**：Q44 问"第一张A之后的下一张是什么牌"（条件概率），本题问"第一张A在哪个位置"（期望值），两者不同。

## Q102: 随机游走 — 一维对称随机游走，从位置 0 出发，首次到达位置 N 的期望步数

**答案：E = N²**

**模型**：每步以 1/2 概率 +1，以 1/2 概率 -1。从 0 出发，首次到达 N 的期望步数？

**推导（可选停止定理）**：

一维对称随机游走是常返的，从 0 出发必然到达 N（概率 1）。

令 Mₜ = Sₜ² - t（Sₜ 为 t 步后的位置），可以验证 Mₜ 是鞅：

$$E[M_{t+1} | \mathcal{F}_t] = E[S_{t+1}^2 - (t+1)] = S_t^2 + 1 - t - 1 = M_t \checkmark$$

由可选停止定理，令 T 为首达 N 的时间：

$$E[M_T] = M_0 \implies E[N^2 - T] = 0 \implies \boxed{E[T] = N^2}$$

**直觉**：扩散方程中位移 ∝ √t，所以走 N 步需要 t ~ N² 步。

**对比赌徒破产（Q25）**：
- Q25 问"0 和 n+m 之间，从 n 出发最终到 n+m 的概率"（概率问题）
- 本题问"从 0 到 N，期望需要多少步"（期望问题）

## Q113: 甲投 201 箭、乙投 200 箭，每箭命中概率均为 0.5，甲命中数严格大于乙的概率

**答案：1/2**

**推导（对称性 + 分解）**：

设甲前 200 箭命中数为 A，乙 200 箭命中数为 B，甲第 201 箭为 X（Bernoulli(1/2)）。

甲总命中 = A + X，要求 A + X > B。

**关键分析**：
- A 和 B 都是 200 次 Bernoulli(1/2) 的总和，分布完全相同，**由对称性 P(A>B) = P(A<B)**。
- 设 P(A>B) = P(A<B) = q，则 P(A=B) = 1 - 2q。

$$P(A + X > B) = P(A > B) + P(A = B) \cdot P(X = 1) = q + (1-2q) \cdot \frac{1}{2} = q + \frac{1}{2} - q = \frac{1}{2}$$

不论 q 是多少，结果恒为 **1/2**。

**直觉**：第201箭的作用是打破平局。平局时以1/2概率决出胜负，非平局时甲乙各已有相同胜率，合计后恰好是1/2。

> **推广**：若甲投 n+1 箭，乙投 n 箭，每箭命中率同为 p（不限于1/2），P(甲>乙) = p（不是1/2！）。
> 仅当 p=1/2 时，P(甲>乙) = 1/2。面试可追问此推广。

## Q114: 连续抛公平硬币，HHT 与 HTT 哪个序列先出现？HHT 先出现的概率是多少

**答案：HHT 先出现，概率为 2/3**

**马尔可夫链建模**：

定义状态（依据"已看到的序列尾部"）：

| 状态 | 含义 |
|------|------|
| S | 初始/上一步是T |
| H | 上一步是H |
| HH | 上两步是HH |
| HT | 上两步是HT |

设 P(HHT先出现 \| 当前状态 X) 为 a, b, c, d：

**列方程**：

- **S**：抛H→H，抛T→S：a = ½b + ½a → **a = b**
- **H**：抛H→HH，抛T→HT：b = ½c + ½d
- **HH**：抛T→HHT（P1赢!），抛H→HH：c = ½·1 + ½c → **c = 1**
- **HT**：抛T→HTT（P2赢!），抛H→H：d = ½·0 + ½b → **d = b/2**

代入 b 的方程：

$$b = \frac{1}{2}(1) + \frac{1}{2}\cdot\frac{b}{2} = \frac{1}{2} + \frac{b}{4} \implies \frac{3b}{4} = \frac{1}{2} \implies b = \frac{2}{3}$$

从初始状态出发：**P(HHT先出现) = a = b = 2/3**

**直觉**：HH状态下，一旦出现T就赢（1/2概率直接赢），而HT状态下必须等T（1/2赢），但下一个H又回到H状态而非HTT起点。HHT对HT状态的"抵抗力"更弱，因此HHT更容易先出现。

> **对比 Q8（已有）**：HH vs HT 的期望步数分别是6和4，但 Q114 是同时博弈谁先出现，是完全不同的问题。

## Q118: 电梯问题 — 8人随机选10层楼中的一层下，电梯期望停靠几次

**答案：E ≈ 5.70 次**

设10层楼，8人各自独立均匀选一层（1/10概率）下楼。

**期望停靠楼层数**（即至少有1人在该楼层下的楼层数期望）：

P(某一层至少有1人下) = 1 - P(所有8人都不选该层) = $1 - \left(\frac{9}{10}\right)^8$

由线性期望：

$$E[\text{停靠次数}] = 10 \times \left[1 - \left(\frac{9}{10}\right)^8\right]$$

计算：$(0.9)^8 = 0.4305$

$$E = 10 \times (1 - 0.4305) = 10 \times 0.5695 = \boxed{5.695 \approx 5.70 \text{ 次}}$$

**一般公式**：n 人，k 层楼，期望停靠次数 = $k\left[1-\left(\frac{k-1}{k}\right)^n\right]$

**与 Q28 的联系**：Q28 给出了"a个球放到b个篮子，期望有球的篮子数 = b[1-(1-1/b)^a]"，本题是其直接应用（a=8人，b=10层）。
