## Q1: X ~ N(0,1)，Y | X ~ N(X, 1)，求 P(Y < 0 且 X > 0)

**答案：1/8**

### 方法一：直接积分（条件概率分解）

$$P(Y < 0,\ X > 0) = \int_0^{\infty} P(Y < 0 \mid X = x) \cdot \varphi(x)\, dx$$

给定 $X = x$，有 $Y \mid X = x \sim N(x, 1)$，所以：

$$P(Y < 0 \mid X = x) = \Phi(-x)$$

代入：

$$P(Y < 0,\ X > 0) = \int_0^{\infty} \Phi(-x) \cdot \varphi(x)\, dx = \int_0^{\infty} (1 - \Phi(x)) \cdot \varphi(x)\, dx$$

令换元 $u = \Phi(x)$，则 $du = \varphi(x)\, dx$；$x=0 \Rightarrow u=\tfrac{1}{2}$，$x \to \infty \Rightarrow u \to 1$：

$$= \int_{1/2}^{1} (1 - u)\, du = \left[u - \frac{u^2}{2}\right]_{1/2}^{1} = \left(1 - \frac{1}{2}\right) - \left(\frac{1}{2} - \frac{1}{8}\right) = \frac{1}{2} - \frac{3}{8} = \boxed{\frac{1}{8}}$$

### 方法二：arcsin 公式（联合正态角度）

将 $Y$ 改写为：$Y = X + Z$，其中 $Z \sim N(0,1)$ 独立于 $X$。

则 $(X, Y)$ 是**联合正态**，参数为：
- $\mu_X = 0,\ \mu_Y = 0$
- $\text{Var}(X) = 1,\ \text{Var}(Y) = \text{Var}(X+Z) = 2$
- $\text{Cov}(X, Y) = \text{Cov}(X, X+Z) = \text{Var}(X) = 1$
- $\rho(X, Y) = \dfrac{1}{\sqrt{1 \cdot 2}} = \dfrac{1}{\sqrt{2}}$

求 $P(X > 0,\ Y < 0)$，令 $V = -Y$，则 $\rho(X, V) = -\dfrac{1}{\sqrt{2}}$，$\theta = \arccos(-\tfrac{1}{\sqrt{2}}) = \tfrac{3\pi}{4}$。

由 arcsin 公式：

$$P(X > 0,\ V > 0) = \frac{1}{4} + \frac{\arcsin(-1/\sqrt{2})}{2\pi} = \frac{1}{4} + \frac{-\pi/4}{2\pi} = \frac{1}{4} - \frac{1}{8} = \frac{1}{8}$$

两种方法一致：$P(Y < 0, X > 0) = \boxed{1/8}$

> **Follow-up 提示：** 求 $P(Y > 0 \mid X > 0)$？= $P(Y>0, X>0) / P(X>0) = (P(X>0) - P(Y<0,X>0)) / (1/2) = (1/2 - 1/8)/(1/2) = 3/4$。

## Q2: 二元正态，求使两个线性组合独立的常数

**题目：** 设 $X, Y$ 联合正态，协方差矩阵 $\Sigma = \begin{bmatrix}1 & 0.5 \\ 0.5 & 2\end{bmatrix}$。求常数 $a$，使得 $U = aX + Y$ 与 $V = X + 2Y$ 独立。

**答案：$a = -9/4$**

对联合正态分布，**独立等价于不相关**，即 $\text{Cov}(U, V) = 0$：

$$\text{Cov}(aX+Y,\ X+2Y) = a\cdot\text{Var}(X) + 2a\cdot\text{Cov}(X,Y) + \text{Cov}(Y,X) + 2\cdot\text{Var}(Y)$$

$$= a \cdot 1 + 2a \cdot 0.5 + 0.5 + 2 \cdot 2 = 2a + 4.5$$

令 $2a + 4.5 = 0$，得 $\boxed{a = -9/4}$。

**关键结论：** 对于联合正态变量，零协方差 $\Leftrightarrow$ 独立（一般分布中此结论不成立）。

## Q3: X ~ N(0,1)，Y ~ N(0,1) 独立，求 Z = X/Y 的分布

**答案：Z 服从标准 Cauchy 分布，无均值和方差**

对任意 $z$，用条件化：

$$P(Z \leq z) = P(X/Y \leq z) = P(X \leq zY, Y > 0) + P(X \geq zY, Y < 0)$$

利用对称性，最终得到 PDF：

$$\boxed{f_Z(z) = \frac{1}{\pi(1+z^2)}}$$

这是参数 $\gamma=1$ 的**标准 Cauchy 分布**。

**Cauchy 分布的特殊性质：**

| 性质 Property | 值 Value |
|---|---|
| 均值 Mean | **不存在**（积分发散）|
| 方差 Variance | **不存在** |
| 中位数 Median | 0 |
| FWHM | 2（见 cheatsheet §4.5）|
| 大数定律 LLN | **不适用**（无有限均值）|
| CLT | **不适用** |

**推广：** 更一般地，若 $X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$ 独立，$\mu_2 \neq 0$，则 $X/Y$ 服从**比率分布（Ratio Distribution）**，退化到 Cauchy 只发生在 $\mu_1=\mu_2=0$ 时。

> **Follow-up 提示：** 如何在实验中检测 Cauchy 分布？样本均值不收敛——多次重复实验，样本均值不会稳定，这是区别于正态的实用检验方法。

## Q4: X ~ N(0,1)，Y ~ N(X, 1)，求条件期望 E[X | Y = y]

**答案：$E[X \mid Y = y] = y/2$**

由 Q1 的联合分布推导，$(X, Y)$ 联合正态：

$$\text{Var}(X) = 1,\quad \text{Var}(Y) = 2,\quad \text{Cov}(X,Y) = 1$$

条件正态公式：

$$E[X \mid Y = y] = \mu_X + \frac{\text{Cov}(X,Y)}{\text{Var}(Y)}(y - \mu_Y) = 0 + \frac{1}{2}(y - 0) = \boxed{\frac{y}{2}}$$

$$\text{Var}(X \mid Y = y) = \text{Var}(X) - \frac{\text{Cov}(X,Y)^2}{\text{Var}(Y)} = 1 - \frac{1}{2} = \frac{1}{2}$$

所以 $X \mid Y = y \sim N(y/2,\ 1/2)$。

**条件正态公式（通用）：**

```
设 (X, Y) 联合正态，则：

X | Y = y  ~  N(μ_X|Y,  σ²_X|Y)

μ_X|Y  = μ_X + ρ·(σ_X/σ_Y)·(y − μ_Y)
                               ↑
                    回归系数 Regression coefficient

σ²_X|Y = (1 − ρ²)·σ²_X       ← 不依赖 y（条件方差是常数）
```

> **Follow-up 提示：** 这正是**线性回归**的概率解释！$E[X \mid Y=y] = \beta_0 + \beta_1 y$ 就是最小二乘回归线，其中 $\beta_1 = \rho \cdot \sigma_X / \sigma_Y$。

## Q5: 高斯混合模型（GMM）的边际均值和方差

**题目：** GMM 有 2 个分量，权重 $\pi_1 = 0.6$，$\pi_2 = 0.4$。分量1：$X \mid Z=1 \sim N(0, 1)$，分量2：$X \mid Z=2 \sim N(3, 1)$。求 $E[X]$ 和 $\text{Var}(X)$。

**答案：$E[X] = 1.2$，$\text{Var}(X) = 3.16$**

**期望（全期望公式 Tower Property）：**

$$E[X] = \sum_k \pi_k E[X \mid Z=k] = 0.6 \times 0 + 0.4 \times 3 = \mathbf{1.2}$$

**方差（Eve's Law 全方差公式）：**

$$\text{Var}(X) = \underbrace{E[\text{Var}(X \mid Z)]}_{\text{组内方差的期望}} + \underbrace{\text{Var}(E[X \mid Z])}_{\text{组间均值的方差}}$$

- 组内方差的期望：$E[\text{Var}(X \mid Z)] = 0.6 \times 1 + 0.4 \times 1 = 1$
- 组间均值的方差：$\text{Var}(E[X \mid Z])$ = 0.6×(0−1.2)² + 0.4×(3−1.2)² = 0.864 + 1.296 = 2.16

$$\text{Var}(X) = 1 + 2.16 = \boxed{3.16}$$

**直觉：** GMM 的总方差 = 组内方差（各高斯自身的展宽）+ 组间方差（各高斯中心位置的差异）。

## Q6: 泊松随机和的期望与方差（Compound Poisson）

**题目：** N 名顾客访问商店，$N \sim \text{Pois}(\lambda)$。每位顾客消费 $X_i \sim N(\mu, \sigma^2)$（独立）。总收入 $S = \sum_{i=1}^N X_i$，求 $E[S]$ 和 $\text{Var}(S)$。

**答案：$E[S] = \lambda\mu$，$\text{Var}(S) = \lambda(\sigma^2 + \mu^2)$**

**期望（Tower Property）：**

$$E[S] = E[E[S \mid N]] = E[N\mu] = \mu E[N] = \boxed{\lambda\mu}$$

**方差（Eve's Law）：**

$$\text{Var}(S) = E[\text{Var}(S \mid N)] + \text{Var}(E[S \mid N])$$

- $\text{Var}(S \mid N=n) = n\sigma^2 \Rightarrow E[\text{Var}(S \mid N)] = \sigma^2 E[N] = \lambda\sigma^2$
- $E[S \mid N=n] = n\mu \Rightarrow \text{Var}(E[S \mid N]) = \mu^2 \text{Var}(N) = \lambda\mu^2$

$$\text{Var}(S) = \lambda\sigma^2 + \lambda\mu^2 = \lambda(\sigma^2 + \mu^2) = \lambda E[X^2]$$

这正是 **Wald 恒等式**的方差版本：$\text{Var}(S) = \lambda \cdot E[X^2]$（当停止时间是 Poisson 时）。

## Q7: 截断正态分布的期望 E[X | X > a]

**题目：** $X \sim N(\mu, \sigma^2)$，求 $E[X \mid X > a]$（单边截断期望）。

**答案：**

$$\boxed{E[X \mid X > a] = \mu + \sigma \cdot \frac{\varphi(\alpha)}{1 - \Phi(\alpha)}}$$

其中 $\alpha = (a - \mu)/\sigma$，$\varphi$ 是标准正态 PDF，$\Phi$ 是标准正态 CDF。

**推导思路（关键步骤）：**

$$E[X \mid X > a] = \frac{\int_a^\infty x \cdot \varphi\left(\frac{x-\mu}{\sigma}\right) \frac{1}{\sigma} dx}{P(X > a)}$$

令 $z = (x-\mu)/\sigma$，利用 $x \cdot \varphi(z) = \mu\varphi(z) + \sigma z \varphi(z)$ 以及 $(z\varphi(z))' = -\varphi(z)$（积分后得 $\varphi(\alpha)$）：

$$E[X \mid X > a] = \mu + \sigma \cdot \frac{\varphi(\alpha)}{1 - \Phi(\alpha)}$$

**特例验证：**
- $a = \mu$（对称点截断）：$\alpha=0$，$E[X \mid X > \mu] = \mu + \sigma \cdot \varphi(0)/(1/2) = \mu + \sigma\sqrt{2/\pi}$
- $a \to -\infty$：$E[X \mid X > -\infty] = \mu + \sigma \cdot 0/1 = \mu$ ✓

**逆 Mill 比率（Inverse Mills Ratio）：** $\lambda(\alpha) = \varphi(\alpha)/(1-\Phi(\alpha))$ 是 Heckman 选择模型的核心项。

## Q8: 顺序统计量 — n 个均匀变量的最小值和最大值

**题目：** $X_1, \ldots, X_n \stackrel{iid}{\sim} U(0,1)$，求最小值 $X_{(1)}$ 和最大值 $X_{(n)}$ 的 PDF 和期望。

**答案：**

**最小值 $X_{(1)} = \min(X_1, \ldots, X_n)$：**

$$F_{(1)}(x) = P(X_{(1)} \leq x) = 1 - P(\text{所有} X_i > x) = 1 - (1-x)^n$$

$$f_{(1)}(x) = n(1-x)^{n-1},\quad x \in [0,1]$$

$$E[X_{(1)}] = \int_0^1 x \cdot n(1-x)^{n-1} dx = \frac{1}{n+1}$$

**最大值 $X_{(n)} = \max(X_1, \ldots, X_n)$：**

$$F_{(n)}(x) = P(X_{(n)} \leq x) = x^n$$

$$f_{(n)}(x) = nx^{n-1},\quad x \in [0,1]$$

$$E[X_{(n)}] = \frac{n}{n+1}$$

**一般顺序统计量 $X_{(k)}$（第 k 小）：**

$$f_{(k)}(x) = \frac{n!}{(k-1)!(n-k)!} x^{k-1}(1-x)^{n-k},\quad E[X_{(k)}] = \frac{k}{n+1}$$

**验证（求和）：** $E[X_{(1)}] + E[X_{(n)}] = \frac{1}{n+1} + \frac{n}{n+1} = 1 = n \cdot E[X_i] \cdot 2/n$... 更直接地：$\sum_{k=1}^n E[X_{(k)}] = \sum_{k=1}^n \frac{k}{n+1} = \frac{n(n+1)/2}{n+1} = n/2 = n \cdot E[U(0,1)]$ ✓

## Q9: 卡方分布 — 正态变量的平方和

**题目：** $Z_1, \ldots, Z_k \stackrel{iid}{\sim} N(0,1)$，求 $Q = Z_1^2 + Z_2^2 + \cdots + Z_k^2$ 的分布、期望和方差。

**答案：$Q \sim \chi^2_k$（自由度为 k 的卡方分布）**

$$E[Q] = k,\qquad \text{Var}(Q) = 2k$$

**推导：**
- 每个 $Z_i^2 \sim \chi^2_1$（自由度为1的卡方）
- 独立卡方之和：$\chi^2_a + \chi^2_b = \chi^2_{a+b}$（利用 MGF 或特征函数）

**与正态的关系：**

| 变换 | 分布 |
|---|---|
| $Z \sim N(0,1)$ | 标准正态 |
| $Z^2$ | $\chi^2_1$ |
| $\sum_{i=1}^k Z_i^2$ | $\chi^2_k$ |
| $\sqrt{\chi^2_k / k}$ (σ未知时的 t 检验分母) | 与 $t_{n-1}$ 相关 |
| $\chi^2_a / \chi^2_b$（归一化后）| F 分布 |

**大样本近似：** 当 $k \geq 30$ 时，$\chi^2_k \approx N(k, 2k)$，或更精确的 $\sqrt{2\chi^2_k} \approx N(\sqrt{2k-1}, 1)$。

**面试应用：** 假设检验中：
- **方差检验**：$\frac{(n-1)s^2}{\sigma^2} \sim \chi^2_{n-1}$
- **卡方拟合检验**：$\chi^2 = \sum (O-E)^2/E \sim \chi^2_{k-1}$

## Q10: X ~ N(0,1)，独立于 Y ~ Exp(1)，求 P(X > Y)

**答案：$P(X > Y) = 1 - \Phi(1/\sqrt{2}) \approx 0.239$**

（更精确地需要数值积分，但可以推导近似值）

**推导：**

$$P(X > Y) = \int_0^\infty P(X > y) \cdot f_Y(y) \, dy = \int_0^\infty (1 - \Phi(y)) \cdot e^{-y} \, dy$$

注意 $P(X > y) = 1 - \Phi(y)$ 仅对 $y \geq 0$ 使用（$y < 0$ 时 $P(X > y) > 1/2$，但 $Y \geq 0$ 所以这里 $y \geq 0$）：

$$= \int_0^\infty \bar{\Phi}(y) e^{-y} dy$$

其中 $\bar{\Phi}(y) = 1 - \Phi(y) = P(Z > y)$。

用 **Mills 比率**和分部积分或数值方法，得到：

$$P(X > Y) \approx \int_0^\infty \bar{\Phi}(y) e^{-y} dy \approx \boxed{0.239}$$

**对比：** 若两者均为 $N(0,1)$，则 $P(X > Y) = 1/2$；指数分布偏右（均值=1），故 $P(X > Y) < 1/2$。

## Q11: 双正态，反向条件分布

**题目：** 设 $\mu_X = 70$，$\mu_Y = 60$，$\sigma_X = 10$，$\sigma_Y = 15$，$\rho = 0.6$，联合正态。已知某学生第一次考了80分，求其第二次考试低于75分的概率。

**答案：$\approx 69.1\%$**

用条件正态公式，$Y \mid X = 80$：

$$\mu_{Y|X=80} = 60 + 0.6 \times \frac{15}{10} \times (80 - 70) = 60 + 9 = 69$$

$$\sigma^2_{Y|X} = (1 - 0.36) \times 15^2 = 0.64 \times 225 = 144 \Rightarrow \sigma_{Y|X} = 12$$

$$P(Y < 75 \mid X = 80) = \Phi\!\left(\frac{75 - 69}{12}\right) = \Phi(0.5) \approx \boxed{0.691}$$

## Q12: 随机变量变换 — U = X + Y，V = X - Y 的独立性

**题目：** $X, Y \stackrel{iid}{\sim} N(0, \sigma^2)$，令 $U = X + Y$，$V = X - Y$。证明 $U$ 和 $V$ 独立，并求各自的分布。

**答案：$U \sim N(0, 2\sigma^2)$，$V \sim N(0, 2\sigma^2)$，且 $U \perp V$**

**分布：**
- $E[U] = 0$，$\text{Var}(U) = \text{Var}(X) + \text{Var}(Y) = 2\sigma^2$
- $E[V] = 0$，$\text{Var}(V) = 2\sigma^2$

**独立性（用协方差）：**

$$\text{Cov}(U, V) = \text{Cov}(X+Y,\ X-Y) = \text{Var}(X) - \text{Cov}(X,Y) + \text{Cov}(Y,X) - \text{Var}(Y)$$

$$= \sigma^2 - 0 + 0 - \sigma^2 = 0$$

由于 $(U, V)$ 是联合正态（$X, Y$ 的线性变换），零协方差即独立，因此 $U \perp V$。

**几何直觉：** 这是 45° 旋转——$(X, Y)$ 坐标系旋转到 $(U, V)$，各向同性的正态在旋转下不变，所以两个方向仍然独立。

**应用：** 这是**Box-Muller 变换**的代数基础，以及正态分布在旋转变换下的不变性（旋转不变性）。

> **Follow-up 提示：** 若 $X, Y$ 不是同方差，结论如何变化？$\text{Cov}(U,V) = \sigma_X^2 - \sigma_Y^2$，只有当 $\sigma_X = \sigma_Y$ 时独立。

## Q13: Z = X² 的分布，X ~ N(0,1)

**题目：** $X \sim N(0,1)$，令 $Z = X^2$，求 $Z$ 的 PDF 和期望。

**答案：$Z \sim \chi^2_1$，$E[Z] = 1$，$\text{Var}(Z) = 2$**

**推导（CDF 法）：**

对 $z > 0$：

$$F_Z(z) = P(X^2 \leq z) = P(-\sqrt{z} \leq X \leq \sqrt{z}) = \Phi(\sqrt{z}) - \Phi(-\sqrt{z}) = 2\Phi(\sqrt{z}) - 1$$

$$f_Z(z) = \frac{d}{dz}[2\Phi(\sqrt{z}) - 1] = 2\varphi(\sqrt{z}) \cdot \frac{1}{2\sqrt{z}} = \frac{\varphi(\sqrt{z})}{\sqrt{z}} = \frac{e^{-z/2}}{\sqrt{2\pi z}}$$

这正是 $\chi^2_1$ 分布的 PDF：$f(z) = \frac{z^{-1/2} e^{-z/2}}{2^{1/2}\,\Gamma(1/2)} = \frac{e^{-z/2}}{\sqrt{2\pi z}}$（因为 $\Gamma(1/2) = \sqrt{\pi}$）。

**期望和方差：**

$$E[Z] = E[X^2] = \text{Var}(X) + (E[X])^2 = 1 + 0 = 1$$

$$E[Z^2] = E[X^4] = 3\sigma^4 = 3 \quad \Rightarrow \quad \text{Var}(Z) = 3 - 1 = 2$$

（注：$E[X^4] = 3$ 对 $N(0,1)$ 是标准结论，由 MGF 的 4 阶导数得到。）

## Q14: 混合分布 — 以概率 p 选择 N(0,1)，以 1-p 选择 N(μ,1)

**题目：** 随机变量 $X$：以概率 $p$ 来自 $N(0,1)$，以概率 $1-p$ 来自 $N(\mu, 1)$。求 $E[X]$、$\text{Var}(X)$，以及该混合分布在何条件下成为**双峰**（bimodal）。

**答案：**

**期望：**
$$E[X] = p \cdot 0 + (1-p) \cdot \mu = (1-p)\mu$$

**方差（Eve's Law）：**
$$\text{Var}(X) = E[\text{Var}(X \mid Z)] + \text{Var}(E[X \mid Z]) = 1 + p(1-p)\mu^2$$

其中组内方差 = 1（两个分量方差均为1），组间方差 = $p \cdot (0-(1-p)\mu)^2 + (1-p) \cdot (\mu - (1-p)\mu)^2 = p(1-p)\mu^2$。

**双峰条件（等权 $p=0.5$）：**

由 cheatsheet §4.6 的结论：两个等方差正态混合成双峰当且仅当两均值间距 $|\mu| > 2\sigma = 2$。

等权时，分布的中点 $\mu/2$ 处是鞍点当且仅当 $|\mu| > 2$（即两峰相距超过 $2\sigma$）。

> **Follow-up 提示：** 这是高斯混合模型 EM 算法的背景：当 $|\mu|$ 较大时，两个分量有清晰分离，EM 容易收敛；当 $|\mu|$ 接近 0 时，两分量几乎重叠，EM 难以分辨。

## Q15: 参数未知的二元正态，计算联合概率

**题目：** $(X, Y)$ 联合正态，$\mu_X = 0$，$\mu_Y = -1$，$\sigma_X^2 = 1$，$\sigma_Y^2 = 4$，$\rho = -1/2$。求 $P(X + Y > 0 \mid 2X - Y = 0)$。

**答案：$\approx 30.85\%$**

令 $U = X + Y$，$V = 2X - Y$（均为联合正态的线性变换，故联合正态）：

**计算各统计量：**

$$E[U] = E[X] + E[Y] = 0 + (-1) = -1$$
$$E[V] = 2E[X] - E[Y] = 0 - (-1) = 1$$
$$\text{Var}(U) = \sigma_X^2 + \sigma_Y^2 + 2\rho\sigma_X\sigma_Y = 1 + 4 + 2(-1/2)(1)(2) = 3$$
$$\text{Var}(V) = 4\sigma_X^2 + \sigma_Y^2 - 2\cdot 2\rho\sigma_X\sigma_Y = 4 + 4 - 2\cdot 2\cdot(-1/2)\cdot 1\cdot 2 = 12$$
$$\text{Cov}(U,V) = \text{Cov}(X+Y, 2X-Y) = 2\sigma_X^2 - \sigma_Y^2 + (2-1)\rho\sigma_X\sigma_Y = 2 - 4 + (-1) = -3$$

**条件分布 $U \mid V = 0$：**

$$E[U \mid V=0] = E[U] + \frac{\text{Cov}(U,V)}{\text{Var}(V)}(0 - E[V]) = -1 + \frac{-3}{12}(0-1) = -1 + \frac{1}{4} = -\frac{3}{4}$$

$$\text{Var}(U \mid V=0) = \text{Var}(U) - \frac{\text{Cov}(U,V)^2}{\text{Var}(V)} = 3 - \frac{9}{12} = 3 - \frac{3}{4} = \frac{9}{4}$$

标准化：$Z = \frac{0 - (-3/4)}{\sqrt{9/4}} = \frac{3/4}{3/2} = \frac{1}{2}$

$$P(U > 0 \mid V=0) = P\!\left(Z > \frac{1}{2}\right) = 1 - \Phi(0.5) \approx 1 - 0.6915 = \boxed{0.3085}$$
